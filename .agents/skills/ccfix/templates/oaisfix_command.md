# 명령어 체계 템플릿

> 중앙 안내: 상세한 에이전트 활용 원칙 및 공통 가이드라인은 `.claude/guides/common_guide.md`의 '2. 에이전트 활용 원칙' 섹션을 참조하십시오.

## 템플릿 설명

이 템플릿은 서브명령어를 사용하는 CLI 도구의 명령어 체계를 정의하기 위한 것입니다.
프로젝트별로 명령어 이름과 서브명령어를 커스터마이즈하여 사용하세요.

## 기본 구조 템플릿

```
[메인명령어] <서브명령어> [대상] [옵션]
```

## 서브명령어 목록 템플릿

| 서브명령어 | 용도 | 예시 |
|-----------|------|------|
| `run` | [기본 실행 용도] | `[메인명령어] run` |
| `preview` | 수정 미리보기 (실제 수정 없이 예상 결과 확인) | `[메인명령어] preview` |
| `test` | 테스트 실행 | `[메인명령어] test` |
| `verify` | 검증만 실행 | `[메인명령어] verify` |
| `rollback` | 마지막 수정 롤백 | `[메인명령어] rollback` |
| `status` | 현재 상태 조회 | `[메인명령어] status` |

## 커스터마이즈 예시 (oofix용)

```
oofix <서브명령어> [대상] [옵션]
```

| 서브명령어 | 용도 | 예시 |
|-----------|------|------|
| `run` | 이슈 자동 수정 실행 | `oofix run` |
| `preview` | 수정 미리보기 (실제 수정 없이 예상 결과 확인) | `oofix preview` |
| `test` | d0003_test.md 시나리오 기반 테스트 실행 | `oofix test` |
| `verify` | 수정 후 검증만 실행 | `oofix verify` |
| `rollback` | 마지막 수정 롤백 | `oofix rollback` |
| `status` | 현재 이슈 상태 조회 | `oofix status` |
