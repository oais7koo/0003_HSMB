## 1. 개요

사용자의 **직전 입력 내용**과 현재 작업 컨텍스트를 요약 표시하는 유틸리티 스킬.

## 2. 명령어

| 명령어 | 설명 |
|--------|------|
| `ccnow` | 직전 사용자 입력 + 현재 컨텍스트 표시 |

## 3. 출력 형식

```markdown
## ccnow — 현재 작업 상황

### 직전 입력
> {사용자의 직전 메시지 원문}

### 컨텍스트
- SP: {현재 서브프로젝트}
- 작업 폴더: {현재 작업 중인 폴더}
- 작업 내용: {진행 중인 작업 요약}
```

> **관련 명령어**: analyze, implement (`.claude/commands/sc/`)

## 4. 동작 규칙

- 사용자의 **바로 직전 메시지**를 그대로 인용하여 표시
- 현재 SP 컨텍스트, 작업 폴더, 진행 중인 작업을 간략히 요약
- 스크립트 불필요 — Claude가 대화 컨텍스트에서 직접 응답
