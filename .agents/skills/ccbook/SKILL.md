## 1. 개요

> ⚠️ **필수**: run 명령 시 반드시 Agent 도구로
> executor(sonnet) 도서 읽기 및 서머리 작성 순서로 위임할 것.
> 도서 전문 직접 읽기 금지 — 메인 컨텍스트 보호.

**목적**: 도서에 대한 상세 서머리를 자동 생성하는 스킬.

- 배치 처리 및 진행 상태 관리

## 3. 도서 서머리 생성

### 3.1 입출력 경로

**입력**:
- Excel 파일: `data/ps1001_박기성/BOOKQAI.xlsx`
  - Q1 시트: 도서명 목록 (A열)
  - Q2 시트: 섹션 주제 (A열 - 7개 섹션)
- 샘플 파일: `data/ps1001_박기성/노화도 설계하는 시대가 온다.md`

**출력**:
- 출력 폴더: `data/ps9003/01_도서/`
- 파일 형식: `[도서명].md`
- 파일 크기: 약 50KB

### 3.2 7개 섹션 구조

| 섹션 | 제목 | 설명 |
|------|------|------|
| 1 | 각 장별로 핵심내용 정리 | 장별 30줄 내외, 핵심키워드별 3줄 내외 |
| 2 | 핵심 통찰과 울림을 주는 구절 30개 | 맥락/의미/원문 + 5분 낭독 스크립트 |
| 3 | 핵심키워드 정리 (30분 서사적 강의용) | 문제제기/원인분석/해결전략/삶의변화 |
| 4 | 강의안 작성 | 시간 배분, 서사적 전개, 청중 참여 |
| 5 | 강의전문 (Full Script) | 4,500~5,500자 분량 연설문 |
| 6 | 구글 슬라이드 작성 가이드 | 15-20장 분량 프레젠테이션 설계 |
| 7 | 플래시카드 및 퀴즈 | 플래시카드 20개, 퀴즈 18문항 |

## 6. AI 실행 규칙

### 6.1 필수 실행 원칙

- 명령어 인식 즉시 첫 번째 도서 섹션 1부터 시작
- 섹션 7까지 완료 -> 자동으로 다음 도서
- 각 섹션 완료 후 파일 저장 -> 다음 섹션 자동 진행
- 전체 서머리를 한 번에 생성하지 않고 **반드시 섹션별로 순차 생성**

### 6.2 절대 금지 사항

- "몇 권만 하시겠습니까?" 질문
- "시범적으로 3권만 해볼까요?" 제안
- 작업 범위 축소 제안
- 간략 버전 제시
- 옵션 선택지 제시

### 6.3 토큰 관리

- 반드시 섹션별로 순차 생성 (전체 한 번에 시 토큰 제한 초과)

## 8. backup/restore 명령어

도서 PDF 파일을 외부 백업 경로로 이동하거나 복원합니다.
대상: `01_book_ko/`, `02_book_en/`, `03_book_ja/`

> **정책**: 백업 경로가 등록되지 않은 컴퓨터에서는 backup/restore 실행 불가. 경로 등록은 `config --path` 로 수동 등록 필요.
