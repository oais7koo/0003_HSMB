## 초록

터널 검사 과정에서 촬영 시스템의 이동으로 인해 발생하는 모션 블러(motion blur)는 딥러닝 기반 균열 검출 성능을 저하시키는 주요 요인이다. 본 연구에서는 체계적인 블러 데이터셋 구축, 4종의 디블러링(deblurring) 알고리즘 개발, 무참조 이미지 품질 평가(NR-IQA) 메트릭 적용, 세그멘테이션(segmentation) 모델 성능 분석을 통해 블러가 균열 검출에 미치는 영향을 정량화하였다. 실험 결과, 블러 강도 증가에 따라 mIoU가 단계적으로 저하되었으며, Richardson-Lucy 디컨볼루션(deconvolution)이 가장 효과적인 복원 성능을 보였다. IQA 메트릭과 세그멘테이션 성능 간 상관관계 분석을 통해 딥러닝 훈련 전 데이터 품질 사전 평가 체계를 제시하였다.

## 1. 서론

터널 안전 관리에서 균열 조기 발견은 구조물 붕괴 방지를 위한 핵심 과제이다. 최근 딥러닝 기반 자동 검출 시스템이 활발히 연구되고 있으나, 현장 환경에서 발생하는 이미지 품질 저하 문제는 충분히 다루어지지 않았다. 특히 검사 차량이나 드론의 이동 중 촬영 시 발생하는 모션 블러는 균열의 경계 정보를 훼손하여 검출 정확도를 크게 낮춘다.

기존 연구들은 고품질 이미지를 전제로 한 알고리즘 개발에 집중하였으며, 블러 환경에서의 성능 분석이나 전처리 전략은 제한적으로 다루어졌다. 본 연구는 실제 터널 검사 환경을 반영한 모션 블러 데이터셋을 구축하고, 다양한 디블러링 기법 적용 후 딥러닝 성능 변화를 체계적으로 분석하여 실무 적용 가능한 품질 관리 및 전처리 전략을 제시하는 것을 목표로 한다.

## 2. 연구 방법

### 2.1 모션 블러 데이터셋 구축

원본 터널 벽체 이미지에 수평 모션 PSF(Point Spread Function)를 적용하여 6단계 블러 레벨(k=0, 10, 20, 30, 40, 50) 데이터셋을 생성하였다. 블러 커널은 h(x,y) = 1/L (0 ≤ x < L, y = 0) 형태로 정의하였으며, 커널 크기 L = 2k + 1로 설정하였다. 블러 이미지는 g(x,y) = (f ⊗ h)(x,y) + n(x,y) 컨볼루션(convolution) 연산으로 생성하였다.

### 2.2 디블러링 알고리즘

본 연구에서는 네 종류의 디블러링 기법을 개발하여 성능을 비교하였다.

Wiener Filter는 MMSE 최적화 기반 주파수 도메인 필터링 방식으로 F_est = [H*/(|H|²+SNR⁻¹)]·G 공식을 적용하였다. SNR은 25로 설정하였으며 처리 속도는 20초/1000장을 나타냈다.

CLAHE + Sharpening은 대비 제한 적응형 히스토그램 평활화(clipLimit=2.0, tileGridSize=8×8)와 Laplacian 에지 강화를 조합한 방식으로 15초/1000장의 처리 속도를 보였다.

Laplacian Sharpening은 2차 미분 기반 에지 강화 방식으로 가장 빠른 처리 속도(10초/1000장)를 제공하였다.

Richardson-Lucy Deconvolution은 베이즈 통계 기반 반복 디컨볼루션 방식으로 f^(k+1) = f^(k)·[h(-x,-y)⊗(g/(f^(k)⊗h))] 공식을 10회 반복 적용하였으며, 처리 속도는 60초/1000장으로 측정되었다.

### 2.3 이미지 품질 평가

무참조 이미지 품질 평가를 위해 5종의 메트릭을 적용하였다. 전통적 방법으로는 BRISQUE(자연 이미지 통계 기반), NIQE(자연성 기반), PIQE(지각 기반)를 사용하였으며, 딥러닝 방법으로는 DBCNN(CNN 기반)과 ARNIQA(Transformer 기반)를 적용하였다. 각 블러 레벨 및 디블러링 변형에 대해 품질 점수를 계산하여 정량적 비교를 수행하였다.

### 2.4 세그멘테이션 성능 평가

Hugging Face의 사전 학습 모델 2종을 활용하여 균열 검출 성능을 평가하였다. SegFormer-b0(nvidia/segformer-b0-finetuned-ade-512-512)는 Transformer 기반 효율적 세그멘테이션 구조로 배치 크기 64, 이미지 크기 512×512, 10 에폭 훈련을 수행하였다. Intel DPT-large(Intel/dpt-large)는 Dense Prediction Transformer 구조로 배치 크기 1-32를 동적으로 조정하였다.

각 블러 레벨별 원본 및 디블러링 데이터에 대해 독립적으로 훈련하여 총 30개 모델을 생성하였으며, mIoU, Pixel Accuracy, Precision, Recall, F1 Score로 성능을 평가하였다.

## 3. 실험 결과

### 3.1 블러 강도에 따른 성능 변화

블러 강도 증가 시 세그멘테이션 성능이 단계적으로 저하되었다. mIoU 기준으로 blur_00 대비 blur_50에서 상당한 성능 감소가 발생하였으며, 이는 모션 블러가 균열 경계 정보를 손상시켜 검출 정확도를 낮춘다는 것을 실증하였다.

### 3.2 디블러링 효과 검증

Richardson-Lucy 디컨볼루션이 가장 높은 복원 성능을 나타냈으며, Wiener Filter, CLAHE+Laplacian, Laplacian 순으로 효과가 나타났다. 중간 강도 블러(blur_20-30)에서는 Wiener Filter가 처리 속도와 품질의 균형점을 제공하였으며, 중증 블러(blur_40-50)에서는 Richardson-Lucy 적용이 필수적임을 확인하였다.

### 3.3 IQA 메트릭과 세그멘테이션 성능 상관관계

5종의 IQA 메트릭과 mIoU 간 Pearson 상관계수를 계산한 결과, 딥러닝 기반 메트릭(DBCNN, ARNIQA)이 전통적 메트릭보다 세그멘테이션 성능을 더 정확하게 예측하였다. 이를 통해 딥러닝 훈련 전 IQA 기반 데이터 품질 사전 평가가 가능함을 확인하였다.

## 4. 결론

본 연구는 모션 블러가 딥러닝 기반 터널 균열 검출에 미치는 영향을 체계적으로 정량화하고, 블러 강도별 최적 디블러링 전략을 제시하였다. 실험 결과를 바탕으로 다음과 같은 3단계 전처리 파이프라인을 제안하였다.

첫째, IQA 기반 사전 필터링 단계에서 품질 메트릭으로 블러 강도를 평가한다. 둘째, 블러 강도별 디블러링 단계에서 경량 블러는 처리를 생략하고, 중간 블러는 Wiener Filter를, 중증 블러는 Richardson-Lucy를 적용한다. 셋째, 딥러닝 학습 단계에서 복원된 데이터로 세그멘테이션 모델을 훈련한다.

향후 연구로는 Blind Deconvolution 기법 적용을 통한 PSF 자동 추정, End-to-End 통합 디블러링-세그멘테이션 모델 개발, Edge Device 배포를 위한 경량화 기술 개발을 계획하고 있다.
