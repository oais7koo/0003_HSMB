## 2. 폴더 명명 규칙

```
crack_{거리}_{ISO}_{속도}
```

| 필드 | 값 | 설명 |
|------|----|------|
| 거리 | d25, d35, d45, d55, d65 | 카메라-터널벽 거리 (단위: × 0.1m, 예: d25 = 2.5m) |
| ISO | ISO100, ISO200, ISO400, ISO800, ISO1600 | 카메라 ISO 감도 |
| 속도 | V60, V80 | 차량(스캐너) 이동 속도 (km/h) |

**예시**: `crack_d45_ISO400_V80` → 거리 4.5m, ISO 400, 속도 80 km/h 조건에서 촬영된 크랙 프레임

## 4. 데이터 특성

### 4.1 모션블러 특성

- **속도(V)**가 높을수록 모션블러 증가 → V80 > V60
- **거리(d)**가 멀수록 단위 픽셀당 이동거리 감소 → 블러 상대적 감소
- **ISO400 조건**: PRD §2.6에 따라 의도적 defocus 조건 — 모션블러 + defocus 복합 블러 발생

### 4.2 크랙 포함 기준

- 실제 터널 스캐닝 영상 전체 프레임 중 **크랙이 육안으로 확인되는 프레임만 추출**
- 동일 크랙 구간을 연속 프레임으로 포함 (frame_XXXXXX.png 번호로 확인 가능)
