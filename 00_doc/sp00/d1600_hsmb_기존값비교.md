## 2. 개요

`data/ps1600_hsmb 기존값비교/` 폴더의 두 xlsx 파일을 비교 분석하여, 논문 제출에 적합한 HSMB 파라미터 세트를 확인하고 이를 기준으로 v1 최적 파라미터 탐색 방향을 정리한다.

## 6. 상관성 분석 결과 (HSMB vs BEW)

| 지표 | 최신 파라미터 | 0715 파라미터 |
|------|-------------|-------------|
| **PLCC** | **-0.9532** | -0.7630 |
| **SROCC** | **-0.7910** | -0.6586 |
| 조건 수 | 20 | 20 |

> 음수 상관: BEW 증가(블러 심함) → HSMB 감소(선명도 낮음) — 방향 정합

**결론: 최신 파라미터(edge_weight=0.8, beta=1.8, JNB=2.5)가 BEW와의 상관성에서 0715 버전보다 유의미하게 우수하여 논문 제출에 적합하다.**

## 7. PRD 사양과의 비교 및 시사점

d0900 검증 결과, 논문 적합 버전(파일1)은 **ps5010 HSMB v1**이며 PRD §4.1 사양과 일치한다.

| 파라미터 | PRD §4.1 사양 | 파일1 (ps5010 v1) | 현재 ps2000 | 비고 |
|---------|-------------|-----------------|------------|------|
| EdgeWeight | 1.5 | **1.5** ✅ | 0.8 ❌ |  |
| β (beta) | 2.0 | **2.0** ✅ | 1.8 ❌ |  |
| JNB | 3 | **3** ✅ | 2.5 ❌ |  |
| p_jnb | 0.63 (cdf[63]) | **cdf[63]** ✅ | cdf[45]≈0.45 ❌ |  |

> **핵심 시사점**:
> - 논문 적합 버전(파일1, PLCC -0.9532)은 PRD §4.1 사양(`ps5010 v1`)과 완전히 일치한다.
> - 현재 `ps2000_통합IQA.py`의 파라미터(ps7310 v2 계통)는 PRD 사양과 다르며 상관성도 열등(-0.7630).
> - **ps2000의 HSMB 파라미터를 PRD 사양(ew=1.5/β=2.0/JNB=3/cdf[63])으로 수정해야 한다.**
> - d5000 파라미터 탐색은 ps5010 v1을 기준 코드로 사용한다.

### 8.2 ablation study 방향 (F001-3 연계)

현재 ps2000 파라미터를 기준점으로 삼아 grid search 범위를 재설정한다.

| 파라미터 | PRD grid 중심 | 권장 grid 중심 (이 분석 기준) |
|---------|-------------|--------------------------|
| EdgeWeight | 1.5 | **0.8** (현재값 포함, 0.5~1.5 범위) |
| β | 2.0 | **1.8** (현재값 포함, 1.0~2.5 범위) |
| JNB | 3 | **2.5** (현재값 포함, 2.0~4.0 범위) |
| p_jnb | 0.63 | **0.45** (현재값 포함, 0.40~0.65 범위) |

> ablation grid는 PRD 사양값과 ps2000 현재값을 **모두 포함**하도록 설계하여
> 두 파라미터 세트 중 어느 쪽이 통계적으로 우월한지 실험으로 입증한다.
