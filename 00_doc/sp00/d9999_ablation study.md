## 1. 개요

Ablation study는 모델이나 알고리즘을 구성하는 요소를 하나씩 제거하거나 값을 바꾸어 보면서, 각 요소가 최종 성능에 얼마나 기여하는지 확인하는 실험 방법이다.

본 프로젝트에서는 HSMB v1 알고리즘의 주요 파라미터인 `EdgeWeight`, `JNB`, `β`가 성능에 미치는 영향을 분석하기 위해 ablation study를 수행한다. 목적은 HSMB v1에서 사용한 기본값이 단순한 경험적 선택이 아니라, 실험적으로 타당한 선택임을 확인하는 것이다.

## 2. Ablation Study의 목적

### 2.1 구성요소의 기여도 확인

알고리즘이 좋은 성능을 보일 때, 어떤 구성요소가 실제로 성능 향상에 기여했는지 분리해서 확인해야 한다. Ablation study는 특정 구성요소를 제거하거나 값을 변경한 뒤 성능 변화를 비교함으로써 각 구성요소의 역할을 검증한다.

### 2.2 파라미터 민감도 분석

파라미터 값이 조금만 바뀌어도 성능이 크게 흔들린다면 알고리즘은 불안정하다고 볼 수 있다. 반대로 일정 범위 안에서 성능이 유지된다면 해당 알고리즘은 파라미터 변화에 비교적 강건하다고 해석할 수 있다.

### 2.3 Reviewer 대응 근거 확보

논문 리뷰에서 특정 파라미터의 이론적 또는 실험적 근거가 부족하다는 지적이 있을 수 있다. 이 경우 ablation study는 파라미터 선택의 정당성을 보여주는 핵심 근거가 된다.

## 3. 본 프로젝트의 Ablation 대상

### 3.1 EdgeWeight

`EdgeWeight`는 평균 gradient 대비 얼마나 강한 edge를 유효 edge로 선택할지 결정하는 파라미터이다. 값이 낮으면 약한 edge까지 포함되어 잡음에 민감해질 수 있고, 값이 높으면 중요한 edge가 누락될 수 있다.

### 3.2 JNB

`JNB`는 Just Noticeable Blur 기준과 연결되는 파라미터로, edge width를 blur probability로 변환할 때 기준점 역할을 한다. 본 프로젝트에서는 JNB 값 변화가 HSMB score와 실제 blur 지표 간 상관성에 어떤 영향을 주는지 확인한다.

### 3.3 β

`β`는 Weibull CDF 기반 blur probability 계산에서 곡선의 형태를 조절하는 파라미터이다. 값이 달라지면 edge width 증가에 따른 blur probability 반응이 완만하거나 급격하게 변한다.

## 4. 실험 설계

### 4.1 기준 알고리즘

Ablation study의 기준은 HSMB v1 알고리즘이다. Track 1 재투고 범위에서는 알고리즘 구조를 변경하지 않고, 기존 v1 구조를 동결한 상태에서 파라미터 변화만 분석한다.

### 4.2 Grid Search

각 파라미터에 대해 여러 후보 값을 정의하고, 가능한 조합을 실행하여 성능을 비교한다. 예를 들어 `EdgeWeight`, `JNB`, `β` 각각 5개 후보 값을 사용하면 총 125개 조합을 평가하게 된다.

### 4.3 평가 지표

각 조합의 성능은 BEW와 HSMB score 사이의 상관성으로 평가한다.

| 지표 | 의미 |
|------|------|
| PLCC | 선형 상관성 |
| SROCC | 순위 상관성 |

PLCC와 SROCC가 높을수록 HSMB score가 실제 blur 수준을 더 잘 반영한다고 볼 수 있다.

## 5. 결과 해석 기준

### 5.1 기본값이 최적 또는 준최적인 경우

현재 기본값인 `EdgeWeight=1.5`, `JNB=3`, `β=2`가 최적 또는 준최적 성능을 보이면, v1 파라미터 선택이 실험적으로 타당하다고 설명할 수 있다.

### 5.2 기본값보다 좋은 조합이 있는 경우

기본값보다 좋은 조합이 발견되면, 재투고 논문에서는 v1의 한계로 정직하게 보고하고 후속 HSMB v2 개선 방향으로 연결한다. Track 1에서는 v1 알고리즘을 동결하므로, 파라미터 변경 자체는 신중하게 판단한다.

### 5.3 파라미터 민감도가 큰 경우

특정 파라미터 변화에 따라 PLCC/SROCC가 크게 흔들리면 해당 파라미터가 HSMB 성능의 핵심 민감 요인임을 의미한다. 이 경우 논문에서는 해당 파라미터의 물리적 의미와 선택 근거를 더 강하게 설명해야 한다.
