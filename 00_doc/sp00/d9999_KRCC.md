## 개요
통계학 및 AI 영상 처리(IQA) 맥락에서 **KRCC**는 Kendall's Rank Correlation Coefficient (켄달 순위 상관계수)를 의미합니다. 그리스 문자 타우($\tau$)를 사용하여 표현하기도 합니다.

앞서 질문하신 PLCC(피어슨), SROCC(스피어먼)와 함께 데이터 분석 및 머신러닝 모델의 평가 지표로 가장 많이 쓰이는 **3대 상관계수** 중 하나입니다.

---

### 1. 핵심 개념: "순위의 방향이 얼마나 일치하는가?"

SROCC(스피어먼)와 마찬가지로 데이터의 실제 수치 대신 순위(Rank)를 비교하는 비모수 통계 방법입니다. 다만, 계산하는 방식에 직관적인 차이가 있습니다.

켄달의 KRCC는 모든 데이터 쌍(Pair)을 2개씩 짝지은 후, "X가 커질 때 Y도 같이 커지는가?"를 전수 조사합니다.

- **부합 쌍 (Concordant pair):** X의 순위가 커질 때 Y의 순위도 같이 커지는 올바른 방향의 쌍
    
- **부합하지 않는 쌍 (Discordant pair):** X의 순위는 커지는데 Y의 순위는 작아지는 반대 방향의 쌍
    

즉, 전체 데이터 쌍 중에서 **'방향이 일치하는 쌍'과 '어긋나는 쌍'의 비율**을 가지고 관계의 강도를 측정합니다.

---

### 2. 값의 범위와 해석

마찬가지로 **$-1$에서 $+1$** 사이의 값을 가집니다.

- **$+1$**: 모든 데이터 쌍의 순위 방향이 완벽하게 일치할 때
    
- **$-1$**: 모든 데이터 쌍의 순위 방향이 완벽하게 정반대일 때
    
- **$0$**: 방향이 일치하는 쌍과 반대인 쌍의 개수가 똑같아 아무런 상관관계가 없을 때
    

---

### 3. SROCC(스피어먼)와 KRCC(켄달)의 차이점

두 지표 모두 순위 기반의 단조적 관계를 보지만, 세부적인 특성이 다릅니다.

- **수학적 민감도:** SROCC는 순위 간의 '거리 차이(제곱)'를 계산에 넣기 때문에 극단적인 순위 변화에 더 민감합니다. 반면 **KRCC는 오직 '방향성(크냐 작냐)'만 따지기 때문에 통계적으로 더 견고(Robust)하고 안정적**입니다.
    
- **표본 크기:** 데이터 개수($n$)가 적을 때는 KRCC가 SROCC보다 더 정확한 경향성을 제시하며 통계적 가설 검정(p-value)을 하기에 더 유리합니다. (다만 데이터가 많아지면 모든 쌍을 조합해야 해서 계산량이 많아집니다.)
    
- **값의 크기:** 일반적으로 같은 데이터를 분석했을 때 **KRCC 값이 SROCC 값보다 다소 작게 나오는 편**입니다. (예: SROCC가 $0.8$일 때 KRCC는 $0.6$ 수준)
    

---

### 4. 종합 요약 (PLCC / SROCC / KRCC)

영상 화질 평가(IQA) 알고리즘이나 AI 예측 모델의 논문을 보면 이 세 가지가 나란히 등장합니다.

1. **PLCC (피어슨):** AI가 예측한 점수가 사람의 실제 평가 점수와 **선형적으로 얼마나 정확하게 일치하는가?**
    
2. **SROCC (스피어먼):** AI가 매긴 화질 순위가 사람이 매긴 순위의 **전체적인 경향성과 얼마나 일치하는가?**
    
3. **KRCC (켄달):** AI가 예측한 데이터 쌍들의 **대소 관계(A가 B보다 우수하다)가 사람의 판단과 얼마나 높은 확률로 일치하는가?**
