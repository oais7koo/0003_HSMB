## 3. 기능 개요

47조건 터널표준영상(`data/ps1204_kict_eSFR/`)에서 BEW-H, BEW-V를 이미지 단위로 산출하고, ps3100이 산출한 NR-IQA 9컬럼 조건별 평균과 PLCC/SROCC/KRCC 상관 분석을 수행한다. 동시에 이방성 지표(BEW_H − BEW_V)로 **ISO=400 의도적 defocus 케이스** 등방성을 검증한다 (PRD §2.6).

BEW는 `common_iqa7.compute_bew_hv()`를 통해 four-direction 분리 산출하며, 상관 계산은 `common_corr.correlation_matrix()`를 공통 모듈로 사용한다. 체크포인트는 조건 완료 단위로 원자적 저장(`_checkpoint.csv.tmp` → rename)하여 재부팅 후 `--resume`으로 이어서 실행 가능하다.

## 9. 처리 절차

### 9.1 조건 폴더 구조

`data/ps1204_kict_eSFR/` 하위 폴더 명명 규칙: `{조도}lx_{ISO}_{셔터속도}us_{속도kmh}`

```
정규식: ^(?P<lux>\d+)lx_(?P<iso>\d+)_(?P<shutter>\d+)us_(?P<speed>\d+)$
예:     15000lx_0640_500us_00, 40000lx_1600_025us_90
```

파싱 불가 폴더(위 정규식 미매칭)는 `[SKIP]` 출력 후 제외한다.

### 9.2 처리 단계

1. `collect_conditions()` — 이미지 디렉토리에서 유효 조건 폴더 목록 수집
2. `--resume` 플래그 시 `_checkpoint.csv` 로드 → `{condition, frame}` 완료 집합 구성
3. 조건 루프: 각 PNG 이미지에 대해 `common_iqa7.compute_bew_hv(img)` 호출 → BEW-H, BEW-V 산출
4. 조건 완료 후 `_checkpoint.csv` 원자적 저장 (`.tmp` → `os.replace`)
5. 전체 완료 후 조건별 평균(BEW-H, BEW-V, 이방성) 집계
6. `data/ps3100/03_조건별평균_*.xlsx` 로드 → 조건명 기준 inner join
7. `common_corr.correlation_matrix()` — IQA 9컬럼 × BEW 2컬럼, PLCC/SROCC/KRCC
8. Scatter plot 3종 생성 (`04_scatter_hsmb`, `05_scatter_all`, `06_anisotropy`)
9. CSV 3종 저장 (`01_bew_per_frame`, `02_bew_per_condition`, `03_correlation`)

### 11.1 BEW 통계 (47조건 평균)

| 지표 | 의미 |
|------|------|
| BEW-H | 수평 방향 Blur Edge Width — 클수록 블러 심함 |
| BEW-V | 수직 방향 Blur Edge Width — 클수록 블러 심함 |
| 이방성 | BEW_H − BEW_V — 양수 = 수평 모션블러 우세 |

> 전체 47조건 상세 수치: `data/ps3101/02_bew_per_condition_2605180906.csv`

**속도 증가에 따른 이방성 증가 패턴 (대표 조건 예시):**

| 조건 | BEW-H | BEW-V | 이방성 |
|------|-------|-------|--------|
| 15000lx_0640_500us_00 (정지) | 1.1702 | 1.3582 | -0.188 |
| 15000lx_0640_500us_30 (30km/h) | 3.0276 | 1.6026 | +1.425 |
| 15000lx_0640_500us_70 (70km/h) | 4.1520 | 1.9752 | +2.177 |
| 40000lx_0640_500us_70 (70km/h) | 4.2911 | 2.1679 | +2.123 |

> 속도 증가 → BEW-H 급증, BEW-V는 완만 증가 → 이방성 확대 = 수평 모션블러 강화 패턴 일치

### 11.2 이방성 분석 — ISO=400 defocus 검증

ISO=400 조건(`40000lx_1600_025us_*`)에서는 이방성이 매우 작음이 관측된다:

| 조건 | BEW-H | BEW-V | 이방성 |
|------|-------|-------|--------|
| 40000lx_1600_025us_00 | 0.9469 | 1.1169 | -0.170 |
| 40000lx_1600_025us_50 | 0.9645 | 1.1197 | -0.155 |
| 40000lx_1600_025us_90 | 0.9899 | 1.1207 | -0.131 |

> 속도 증가에도 이방성이 -0.17 ~ -0.13 수준으로 거의 변화 없음. defocus(등방성 블러) 지배 조건임을 확인 → PRD §2.6 "의도적 defocus = isotropic 블러" 정합.
