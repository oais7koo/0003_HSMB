## 2. 기능 개요

47조건 터널표준영상에서 측정한 BEW-H, BEW-V를 F002-1의 NR-IQA 7종 스코어와 상관분석(PLCC/SROCC, scatter)하여 **각 메트릭의 모션블러 특화도**를 정량 비교. 동시에 ISO=400 조건의 이방성 분석으로 **의도적 defocus 케이스**를 검증 (PRD §2.6).

## 3. 요구사항

| ID | 요구사항 | 우선순위 | 출처 |

| R01 | 47조건 BEW-H, BEW-V 일괄 측정 | Must | PRD F14b |
| R02 | NR-IQA 7종 × {BEW-H, BEW-V} scatter 14종 생성 | Must | PRD F14b |
