## 3. 기능 개요

청송 터널 MTF 이동촬영 데이터셋(`ps1010_chungsong_MTF`, 50조건 frame 507장) 전체에 대해 **d0100 표준 7종 NR-IQA** 스코어를 `ps0010_new_iqa_lib.py`로 일괄 산출한다. NIQE·BRISQUE는 pyiqa 기본/MATLAB 두 변형을 모두 산출하므로 출력은 **9컬럼**이다. PRD v06 §2.7에서 신규 정의한 E3 실험군의 첫 작업으로, 현장 50조건 실측 데이터셋의 IQA 분포 기준선을 확보한다.

IQA 산출은 정식 라이브러리 `ps0010_new_iqa_lib.py`(HSMB 자체구현 + CPBD PyPI + pyiqa 5종)를 사용하며, batch runner는 이 라이브러리를 import해 50조건을 순회한다.

## 5. 입출력 정의

### 5.1 입력

| 항목 | 타입 | 설명 | 필수 |
|------|------|------|------|
| `data/ps1010_chungsong_MTF/` | folder | 50개 조건 폴더, 각 폴더 직속 `frame_*.png` (총 507장) | Y |
| `src/ps0010_new_iqa_lib.py` | Python | 7종 IQA 정식 산출 라이브러리 (`compute_all_from_path()`) | Y |

### 5.2 출력

| 항목 | 타입 | 설명 |
|------|------|------|
| `src/ps3010_e3_1_ps1010_iqa.py` | Python | ps1010 50조건 일괄 실행 진입점 (ps0010 import) |
| `data/ps3010/01_전체데이터_{dsr}.xlsx` | XLSX | 이미지별 9컬럼 스코어 + 조건 메타 (507행) |
| `data/ps3010/02_그룹별통계_{dsr}.xlsx` | XLSX | 조건별 mean/std/min/max (50조건) |
| `data/ps3010/03_그룹별평균_{dsr}.xlsx` | XLSX | 조건별 평균 + best_group 행 |
| `data/ps3010/04_histogram_{dsr}.png` | PNG | 9종 메트릭 히스토그램 |
| `00_doc/sp00/d0043_e3_1_ps1010.md` | MD | 결과 요약 + 조건별 분포 표 + 히스토그램 |

### 5.3 출력 스키마 (`data/ps3010/01_전체데이터_{dsr}.xlsx`)

```
condition, distance_m, speed_kmh, iso, frame, is_defocus,
hsmb, cpbd, niqe, niqe_matlab, piqe, brisque, brisque_matlab, dbcnn, arniqa
```

- `condition`: 조건 폴더명 (예: `low_4.5m_60km_4.5m_ISO400`)
- `is_defocus`: ISO=400 조건이면 1, 아니면 0
- IQA 9컬럼: ps0010 `METRIC_COLUMNS` — NIQE·BRISQUE는 기본/MATLAB 2변형 산출
- 공통 규칙: d0100 §6.4 — UTF-8, 소수점 4자리(`.4f`), 결측 `NaN`

## 6. 제약조건 / 예외처리

| 상황 | 처리 방식 |
|------|----------|
| GPU OOM (50조건 일괄) | 조건 단위 분할 + 매 조건 후 `torch.cuda.empty_cache()` |
| 폴더명 파싱 실패 | 경고 로그 + 해당 조건 스킵 (수동 검토용 리스트) |
| 일부 NR-IQA 모델 로드 실패 | ps0010 `compute_all()`이 해당 컬럼만 `NaN` 처리, 나머지 진행 |
| `Results/` 하위 PNG 혼입 | 조건 폴더 직속 `frame_*.png`만 glob, `Results/` 명시 제외 |
| ISO=400 조건 (의도적 defocus) | `is_defocus` 플래그 컬럼 표시 (PRD §2.6, F003-2 연계) |
| ps0010 의존성 미설치 (pyiqa/cpbd) | GPU 환경 `uv sync` + ps0010 동작 검증 선결 (d0004 TODO) |
