## 3. 알고리즘 개요 (PRD §4.1 동결 파라미터)

```
입력: 2D grayscale numpy array
  1. Sobel gradient (gx, gy, gmag)
  2. 블록 분할 (block_size × block_size)
  3. AEP 선택: gmag(p) > mean(gmag_block) × edge_weight
  4. 그래디언트 방향 4-bin 양자화 (0°/45°/90°/135°)
  5. 각 AEP에서 수직 방향 프로파일 추출 (walk_radius, 3점 포물선 서브픽셀)
  6. 엣지 폭 w → 블러 확률: P_blur = 1 - exp(-(w/jnb)^beta)
  7. HSMB = mean(P_blur <= p_jnb) — P_jnb 이하 AEP 비율
출력: float [0.0, 1.0]  (높을수록 선명)
```

## 4. 공개 API

### 4.1 `compute_hsmb()`

```python
def compute_hsmb(
    image: np.ndarray,
    block_size: int = 64,
    edge_weight: float = 1.5,
    jnb: float = 3.0,
    beta: float = 2.0,
    p_jnb: float = 0.63,
    walk_radius: int = 8,
) -> float
```

| 파라미터 | 기본값 | 설명 |
|---------|--------|------|
| image | — | 2D grayscale ndarray |
| block_size | **64** | 블록 크기 (px) |
| edge_weight | **1.5** | AEP 임계 승수 (PRD §4.1) |
| jnb | **3.0** | Just-Noticeable-Blur 기준폭 (px) |
| beta | **2.0** | Weibull 형상 파라미터 |
| p_jnb | **0.63** | CDF 쿼리 포인트 |
| walk_radius | 8 | 수직 프로파일 절반 길이 |

**반환**: HSMB 스코어 `[0.0, 1.0]`. 유효 AEP 없으면 `0.0`.

**예외**: `ValueError` — image가 2D가 아닌 경우.

## 5. ps2000/ps2010 내장 계산기 vs ps2020

| 항목 | ps2000 내장 | ps2010 내장 (`_HSMBv1Calculator`) | ps2020 `compute_hsmb()` |
|------|-----------|----------------------------------|------------------------|
| 엣지 프로파일 | profile_length=10, threshold_ratio 기반 종료 | 동일 | **walk_radius 기반 monotonicity walk + 3점 포물선 서브픽셀** |
| 방향 처리 | 4방향 단순 최대강도차 | 동일 | **4-bin 양자화 (각도 기반)** |
| Lap+FFT 융합 | ✅ (v2 계통 잔존) | ❌ 없음 | ❌ 없음 |
| 알고리즘 충실도 | 낮음 | 중간 (ps5010 인라인) | **가장 높음** |
| 코드 근거 | ps7310 v2 계통 | ps5010_HSMB_v1.py 직접 이식 | 논문 알고리즘 재구현 |

> **권장 사용**: 논문 제출용 정식 HSMB 값은 `ps2020.compute_hsmb()` 사용.

## 6. 알고리즘 상세 — ps2010 대비 차이점

### 6.1 엣지 프로파일 추출 방식

| 구분 | ps2010 (`_HSMBv1Calculator`) | ps2020 (`compute_hsmb`) |
|------|---------------------------|------------------------|
| 방법 | `threshold_ratio × center` 기반 종료 | monotonicity walk (단조성 기반) |
| 서브픽셀 | 선형 보간 | **3점 포물선 피팅** |
| 프로파일 길이 | PROFILE_LENGTH=10 고정 | walk_radius=8 (양방향 탐색) |

### 6.2 방향 결정

ps2010: 인접 4방향 중 강도차 최대 방향 (heuristic)
ps2020: `arctan2(gy, gx)` → 4-bin 양자화 (0/45/90/135°) — 이론적으로 더 정확
