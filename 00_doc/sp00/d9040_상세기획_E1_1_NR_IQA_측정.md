## 2. 기능 개요

터널표준영상(eSFR 차트, 47조건) 전체에 대해 **HSMB + 비교 NR-IQA 6종(7종 합계)** 스코어를 일괄 산출하여 E1-2 상관 분석의 입력 데이터를 생성. README가 명시한 "다음 작업".

## 5. 제약조건 / 예외처리

| 상황 | 처리 방식 |
|------|----------|
| GPU OOM (47조건 일괄) | 조건 단위 분할 + 매 조건 후 torch.cuda.empty_cache() |
| 폴더명 파싱 실패 | 경고 로그 + 해당 조건 스킵 (수동 검토용 리스트) |
| 일부 NR-IQA 모델 로드 실패 | 해당 메트릭만 NaN 처리, 나머지 진행 |
| ISO=400 조건 (의도적 defocus) | 별도 플래그 컬럼 표시 (PRD §2.6, F003-2와 연계) |
