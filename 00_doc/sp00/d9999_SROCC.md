## 개요
통계학에서 **SROCC**는 Spearman's Rank-Order Correlation Coefficient (스피어먼 순위 상관계수)를 의미합니다. 줄여서 그냥 스피어먼 상관계수($\rho$ 또는 $r_s$)라고도 많이 부릅니다.

일반적으로 잘 알려진 피어슨(Pearson) 상관계수가 데이터의 '실제 값'을 가지고 '선형적인 관계'를 측정한다면, SROCC는 데이터의 '순위(Rank)'를 가지고 '단조적인 관계(Monotonic 관계)'를 측정한다는 점이 가장 큰 특징입니다.

---

### 1. 핵심 개념: 단조적 관계(Monotonic Relationship)란?

SROCC는 한 변수가 증가할 때 다른 변수가 **일관되게 증가하거나 감소하는 경향**이 있는지를 봅니다. 직선(선형) 모양이 아니더라도, 두 변수가 같은 방향으로 움직이기만 하면 높은 점수를 줍니다.

- **피어슨 vs 스피어먼 비교:**
    
    - 곡선 형태로 완벽하게 증가하는 데이터가 있다면, 피어슨은 직선이 아니기 때문에 $1$보다 낮은 값이 나오지만, SROCC는 순위가 완벽히 일치하므로 $+1$이 나옵니다.
        

---

### 2. SROCC의 수식적 이해

데이터의 개수가 $n$개이고, 각 데이터의 두 변수 순위 차이를 $d_i$라고 할 때 수식은 다음과 같습니다.

$$SROCC = 1 - \frac{6 \sum d_i^2}{n(n^2 - 1)}$$

- **$d_i$**: $i$번째 데이터의 (X 변수 순위) $-$ (Y 변수 순위)
    
- **$n$**: 전체 샘플 수
    

> **쉽게 이해하기:**
> 
> 만약 모든 데이터의 X 순위와 Y 순위가 완전히 똑같다면, 순위의 차이인 $d_i$는 전부 $0$이 됩니다. 분자가 $0$이 되므로 최종 SROCC 값은 $1$이 됩니다. 반대로 순위가 완전히 뒤집혀 있다면 $-1$이 됩니다.

---

### 3. 주요 특징과 장점

- **이상치(Outlier)에 강함 (로버스트성):**
    
    실제 데이터 값 대신 '순위'를 쓰기 때문에, 혼자서 말도 안 되게 큰 값을 가진 이상치가 하나 껴있더라도 전체 결과가 왜곡되지 않습니다. (피어슨은 이상치에 크게 휘둘립니다.)
    
- **비선형 관계 파악 가능:**
    
    지수적으로 증가하거나 로그 형태로 증가하는 등, 직진은 아니지만 한쪽이 커질 때 같이 커지는 관계를 정확히 잡아냅니다.
    
- **서열 척도(Ordinal Data) 사용 가능:**
    
    "매우 만족(5점), 만족(4점)..." 처럼 정확한 수치 데이터가 아닌, 순서만 의미가 있는 설문조사 데이터 등에도 바로 적용할 수 있습니다.
    

---

### 4. 언제 주로 사용할까?

- **데이터가 정규분포를 따르지 않을 때:** 데이터가 한쪽으로 심하게 치우쳐 있을 때 사용하는 대표적인 비모수 통계 방법입니다.
    
- **머신러닝 / 영상 처리 (IQA):** AI 모델이 예측한 점수의 순위와, 사람이 평가한 주관적 점수(MOS)의 순위가 얼마나 일치하는지 비교할 때 표준 지표로 사용됩니다.
    
- **의학 / 바이오:** 환자의 증상 호전 순위와 약물 투여량 순위 간의 관계를 분석할 때처럼, 정밀한 수치화가 어려운 변수들을 다룰 때 유용합니다.
