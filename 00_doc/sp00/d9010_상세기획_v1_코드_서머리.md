## 2. 기능 개요

`data/01_기존/`에 누적된 74개 v1 개발 스크립트를 인벤토리화하여 **HSMB v1 final 버전을 식별**하고, 해당 코드를 `src/hsmb/v1.py`로 모듈화하여 재투고 논문 실험 재현성을 확보한다.

## 3. 요구사항

| ID | 요구사항 | 우선순위 | 출처 |
|----|---------|---------|------|
| R01 | 74개 스크립트 인벤토리 작성 (파일명, mtime, 크기, 역할 분류) | Must | PRD F01 |
| R02 | HSMB v1 final 버전 식별 (git log + mtime + 결과 CSV 일치성 교차) | Must | plan R02 |
| R03 | 식별된 final 코드를 `src/hsmb/v1.py` 단일 모듈로 정리 | Must | plan §5.4 |
| R04 | v1 단위테스트 작성 (regression: 50조건 결과 PLCC=0.9354 재현 검증) | Must | PRD §10 |
