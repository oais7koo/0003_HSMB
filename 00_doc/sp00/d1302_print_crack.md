# d1302 — 프린트 크랙 이미지 데이터셋 (ps1302_print_crack)

## 1. 개요

| 항목 | 내용 |
|------|------|
| **데이터셋명** | ps1302_print_crack |
| **경로** | `data/ps1302_print_crack/` |
| **출처** | 터널 벽면에 크랙 패턴을 인쇄한 종이(프린터 출력물)를 부착하여 촬영한 이미지 |
| **총 이미지 수** | 50장 (조건당 1장) |
| **이미지 형식** | PNG |
| **카메라** | cam1 |
| **용도** | 제어된 환경에서의 크랙 검출 평가 — 실제 크랙 vs 프린트 크랙 비교 기반 |

---

## 2. 촬영 방법

실제 터널 내부 촬영 조건과 동일한 설정에서, 터널 벽면에 크랙 패턴을 인쇄한 출력물을 부착하고 MTSS 스캐너로 촬영하였다. 이를 통해 크랙의 존재/위치는 완전히 통제된 상태에서 조명·속도·ISO·거리에 따른 모션블러 영향을 순수하게 분리하여 평가할 수 있다.

---

## 3. 파일 명명 규칙

```
cam1_{속도}_{ISO}_{거리}[_x].png
```

| 필드 | 값 | 설명 |
|------|----|------|
| 카메라 | cam1 | 카메라 번호 |
| 속도 | v60, v80 | 스캐너 이동 속도 (km/h) |
| ISO | iso100, iso200, iso400, iso800, iso1600 | 카메라 ISO 감도 |
| 거리 | d25, d35, d45, d55, d65 | 카메라-벽면 거리 (× 0.1m, 예: d25 = 2.5m) |
| `_x` | 접미사 (선택) | 사용 불가 이미지 (크랙 미식별·품질 불량 등) |

**예시**: `cam1_v80_iso400_d45.png` → cam1, 80 km/h, ISO 400, 거리 4.5m
