### IV. 제안하는 무참조 이미지 품질 평가

#### A. HSMB 메트릭의 동기와 기존 NR-IQA의 한계

터널 점검과 같은 고속 영상 응용에서 이미지 품질 유지와 결함 검출 신뢰성 향상을 위해서는 MB에 대한 정확한 평가가 필수적이다. 그러나 기존 NR-IQA 메트릭 대부분은 일반적 품질 평가 목적으로 개발되어, 고속 환경에서의 MB 특성에 최적화되어 있지 않다.

예컨대 BRISQUE \[45\]와 NIQE \[62\]는 자연 장면 통계(NSS) 모델에 기반한다. 이들은 JPEG 압축, 잡음, 일반 블러와 같은 왜곡에는 잘 동작하지만, 방향성이 있고 강도가 변하는 MTSS 환경의 MB 통계 특성을 포착하지는 못한다. 이들 모델은 미리 정의된 왜곡 유형의 데이터셋으로 학습되므로, 고유한 블러 특성을 갖는 환경에 대한 적응력이 제한적이다.

PIQE \[64\]는 블록 기반 특징 추출과 기계 학습 분류로 이미지 품질을 평가한다. 그러나 고속 환경의 MB는 공간적으로 비균일하고 방향 의존적인 경우가 많아, 블록 기반 기법은 세밀한 공간 변동과 방향성 블러 패턴을 표현하기 어렵고, 결과적으로 블러 강도 추정 효과가 떨어진다.

CPBD \[64\]는 Canny 필터로 검출한 엣지에서 휘도 변화를 분석하여 블러를 평가한다. 이는 인프라 이미지의 균열과 같은 구조적 결함 검출에 적합하다. CPBD는 JND 조건을 초과하는 대비에서 엣지 주변 인지 가능 최소 블러 수준을 규정하는 JNB 개념을 도입하고, 누적 확률을 이용해 전역적으로 균일한 블러를 주로 평가한다 \[65\]. 그러나 국소적이거나 복잡한 블러 패턴이 포함된 이미지에서는 성능이 저하될 수 있다 \[66\].

종합하면, BRISQUE·NIQE·PIQE 등 일반 목적 NR-IQA 메트릭은 HSMB 특성을 충분히 대표하지 못한다. CPBD는 일부 구조 점검 상황에 적용 가능하지만 엣지 검출 의존성으로 특정 조건에서 정확도가 떨어질 수 있다. MTSS와 같은 고속 환경의 MB는 일반 왜곡과 크게 달라, 해당 조건에 특화된 전용 NR-IQA 메트릭 개발이 필요하다.

이러한 한계를 해결하기 위해, 본 연구는 고속 모션블러의 고유 특성을 포착하여 MTSS 환경에서의 이미지 품질 평가 신뢰성을 향상시키는 새로운 NR-IQA 기법인 **HSMB 메트릭**을 제안한다. 제안 메트릭은 엣지 변화에 대한 민감도를 높이고 블러 강도를 더 강건하게 추정하여, 궁극적으로 MB로 인한 결함 검출 오류를 줄이는 것을 목표로 한다.

#### B. HSMB 메트릭 개요

HSMB 메트릭은 **능동 엣지 픽셀(AEP)** 접근에 기반한다. Canny 필터 등 전통적 엣지 검출기를 사용하는 CPBD와 달리, HSMB는 엣지 정보가 불완전하거나 잘못 식별되어 발생하는 블러 추정 오류를 줄이기 위해 엣지 검출기에 의존하지 않고 이미지 그래디언트 크기를 직접 분석하여 블러를 정량화한다. 이 설계는 터널 이미지의 균열·잡음 영향을 감소시키며, 블러 강도 측정의 안정성을 높인다. HSMB는 엣지 영역의 급격한 강도 변화로 블러를 식별하고, 해당 영역의 블러 확산을 정량화하여 최종 점수를 생성한다. HSMB 메트릭의 알고리즘 블록 다이어그램은 Fig. 9에 제시되어 있다.

HSMB 처리 절차는 다음과 같다:

1. 국소 이미지 특성을 반영하기 위해 입력 이미지를 64 × 64 블록으로 분할한다. 이 분할은 이미지 전반에 걸친 미세 엣지 검출을 가능하게 하며, BRISQUE와 같은 NR-IQA 기법의 블록 기반 분석 관행에 부합한다.

2. 블록 내 각 픽셀에 대해 Sobel 연산자로 수직(x)·수평(y) 방향 그래디언트를 계산하고, 그래디언트 벡터 크기를 구해 엣지 강도를 산출한다. 블록 내 모든 픽셀의 평균 엣지 강도에 가중치 계수(Edge Weight)를 곱해 식 (1)의 임계값 T를 결정한다. 엣지 강도가 T보다 큰 픽셀을 블러 평가용 AEP로 선정한다. 터널 이미지 실험을 기반으로 Edge Weight는 1.5로 설정하여, 그래디언트 크기 상위 약 75 %가 AEP로 분류되도록 하였다(Fig. 10; 이론 근거는 §IV-D).

$$T\;=\;w_{e}\cdot\langle G\rangle,\qquad w_{e}=1.5\tag{1}$$

여기서 $\langle G\rangle = \frac{1}{|B|}\sum_{p\in B} G(p)$는 64 × 64 블록 $B$ 내 Sobel 그래디언트 크기의 평균이며, $G(p)=\sqrt{G_{x}(p)^{2}+G_{y}(p)^{2}}$, $w_{e}$는 Edge Weight(기본값 1.5)이다.

3. 각 AEP의 엣지 방향은 주변 픽셀 중 강도 변화가 최대인 방향으로 결정된다. CPBD가 라디안 값을 연속적으로 계산하는 것과 달리, 제안 기법은 4개 주방향(0°, 90°, 180°, 270°)으로 제한하여 계산을 단순화하며, 선택 사항으로 대각 방향(45°, 135°, 225°, 315°)을 추가로 포함할 수 있다.

4. 각 AEP에서 양자화된 그래디언트 방향을 따라 엣지에 직교하는 강도 프로파일을 샘플링하며, AEP를 중심으로 양방향 $2r+1$ 픽셀(기본 walk-radius $r=8$, 즉 17개 샘플) 대칭 윈도우를 사용한다. 기존 ESF 기반 기법이 강도 변화의 20 %~80 % 구간을 엣지 폭으로 정의하는 것과 달리(Fig. 11(a)), 제안 기법은 **단조성 walk**와 서브픽셀 **3-점 포물선 정밀화**를 결합하여 엣지 천이를 국소화한다(Fig. 11(b)). 프로파일 중심에서 시작해 양방향으로 walk를 진행하며, 각 방향은 첫 비단조 단계 — 엣지 천이의 양 끝을 bracket하는 국소 강도 극값 — 에서 정지한다. 두 끝점은 이산 프로파일 값에 대한 3-점 포물선 보간으로 서브픽셀 정밀도까지 정밀화되며, 엣지 폭 $w(e_i)$은 두 정밀화된 끝점 사이의 거리로 정의된다. 이 단조성 기반 국소화는 낮은 강도 변화가 블러로 오인될 수 있는 ESF 기반 기법 대비 잡음 강건성이 향상되며, 포물선 서브픽셀 정밀화는 정수 픽셀 격자 이하의 메트릭 변별력을 보존한다.

5. 다음으로 주어진 AEP $e_i$에 대한 블러 확률 $P_{\text{blur}}(e_i)$을 식 (2)로 계산한다. §IV-D의 이론적 분석에 근거하여 JNB = 3, Weibull 형상 모수 $\beta = 2$(Rayleigh)로 모든 실험에서 고정하였다.

$$P_{\text{blur}}(e_{i})\;=\;1-\exp\!\left[-\left(\frac{w(e_{i})}{\text{JNB}}\right)^{\beta}\right]\tag{2}$$

여기서 $w(e_i)$는 단계 4의 EVP 보간으로 얻은 AEP $e_i$의 서브픽셀 엣지 폭이다. 식 (2)는 척도 모수 $\text{JNB}$와 형상 모수 $\beta$를 갖는 Weibull 누적 분포 함수이며, $\beta = 2$에서 Rayleigh 형으로 환원된다(이론적 정당화는 §IV-D).

블러 확률 $P_{\text{blur}}(e_i)$은 0과 1 사이 값을 가진다. 최종 HSMB 점수는 식 (3)으로 계산되며, 지각 임계값 $P_{\text{jnb}}$는 0.63으로 설정된다.

$$\text{HSMB}\;=\;F_{P_{\text{blur}}}(P_{\text{jnb}}),\qquad P_{\text{jnb}}=0.63\tag{3}$$

여기서 $F_{P_{\text{blur}}}(\cdot)$은 이미지 전체에서 모든 AEP의 블러 확률 집합 $\{P_{\text{blur}}(e_i)\}_{i=1}^{N_{\text{AEP}}}$의 경험적 누적 분포 함수(CDF)이며, $F_{P_{\text{blur}}}(P_{\text{jnb}})$는 블러 확률이 $P_{\text{jnb}}$ 이하인 AEP의 비율을 의미한다. 실무적으로 HSMB 값은 $P_{\text{blur}}$ 경험 분포로부터 CDF를 구성하고 $P_{\text{jnb}} = 0.63$에서 평가하여 얻는다. 이 정의에 의해 HSMB는 [0, 1] 구간 값을 가지며, 값이 클수록 더 선명한 이미지를 의미한다 — 선명한 이미지는 $P_{\text{blur}}$가 0 근처에 집중되어 임계값 이하에 더 많은 AEP가 분포하기 때문이다.

§IV-B의 다섯 단계와 식 (1)~(3)을 직접 재현 가능한 단일 절차로 통합하여 Algorithm 1에 정리한다.

**Algorithm 1: HSMB 메트릭**

```
입력 :  영상 I (그레이스케일 또는 RGB의 휘도 채널)
        하이퍼파라미터 (w_e, JNB, β, P_jnb) = (1.5, 3, 2, 0.63)
        블록 크기 N_b = 64
출력 :  HSMB 점수 (∈ [0, 1], 값이 클수록 선명)

 1: AEP ← ∅                                   // 능동 엣지 픽셀 집합
 2: 영상 I를 비중첩 N_b × N_b 블록 {B_k}로 분할
 3: 각 블록 B_k에 대해
 4:    G_x, G_y ← B_k의 Sobel 그래디언트
 5:    G(p) ← sqrt(G_x(p)² + G_y(p)²)         (모든 p ∈ B_k에 대해)
 6:    ⟨G⟩_k ← B_k 내 G(p)의 평균
 7:    T_k ← w_e · ⟨G⟩_k                      // 식 (1)
 8:    AEP ← AEP ∪ {p ∈ B_k : G(p) > T_k}
 9: 반복 종료
10: P ← ∅                                     // 블러 확률 집합
11: 각 e_i ∈ AEP에 대해
12:    그래디언트 방향 θ(e_i)을 4 bin {0°, 45°, 90°, 135°}으로 양자화
13:    e_i 중심 직교 프로파일 2r+1 픽셀 샘플링 (r = 8)
14:    중심에서 양방향으로 walk, 첫 비단조 단계에서 정지
15:    좌/우 끝점을 3-점 포물선 보간으로 서브픽셀 정밀화
16:    w(e_i) ← |right_refined − left_refined|
17:    P_blur(e_i) ← 1 − exp(−(w(e_i)/JNB)^β) // 식 (2)
18:    P ← P ∪ {P_blur(e_i)}
19: 반복 종료
20: F ← P의 경험적 CDF
21: HSMB ← F(P_jnb)                            // 식 (3); HSMB ∈ [0, 1]
22: HSMB 반환
```

참고 구현은 프로젝트 저장소의 `scripts/hsmb_metric.py`에 제공된다. 이미지당 FLOPs와 추론 시간은 구현팀의 별도 벤치마킹 환경에서 측정되며, 런타임 특성 분석이 완료되는 시점에 보고한다.

#### C. 민감도 분석 (Sensitivity Analysis)

HSMB 메트릭은 Table 2에 요약된 3개 주요 파라미터에 의존하며, 최종 출력은 이들의 결합된 영향으로 결정된다. 각 파라미터의 HSMB 출력에 대한 기여를 평가하기 위해 Sobol 민감도 지수 \[67\]를 이용한 민감도 분석을 수행하였다.

Table 3(민감도 분석 결과)는 **Edge Weight(0.223), Direction(0.219), Threshold Ratio(0.208)**의 TSI를 보고한다. 모든 파라미터가 0.2보다 큰 TSI와 0.01 미만의 신뢰 구간을 보여, 각 파라미터가 HSMB 출력에 유의미한 영향을 미치며 추정이 통계적으로 안정적임을 확인하였다. 특히 Edge Weight가 가장 높은 민감도를 보였는데, 이는 AEP 선정에서의 중심적 역할을 반영한다 — AEP 식별이 HSMB 점수에 기여하는 픽셀을 직접 결정하므로, Edge Weight가 최종 메트릭 값에 상당한 영향을 미친다.

#### D. 파라미터 선정과 이론적 근거 *(신규, 2026-04-22 — R04 대응)*

HSMB 메트릭은 세 개의 하이퍼파라미터 — Edge Weight, JNB, β — 를 가지며, 기본값은 각각 1.5, 3, 2 이다. 각 기본값은 자연 이미지 엣지에 대한 잘 정립된 통계·지각 모델에 근거한다. §IV-C의 Sobol 민감도 분석은 세 파라미터가 HSMB 출력에 유의미한 영향을 미침을 보였으며, 본 절은 각 기본값이 *왜* 이론적으로 타당한지를 설명한다. Section IV-E에서는 이에 대한 상관분석 기반 실험적 검증을 제공한다.

**1) Edge Weight (기본값 = 1.5): 자연 이미지 그래디언트의 GGD 모델**

자연 이미지의 그래디언트 크기는 Generalized Gaussian Distribution(GGD) \[75\]로 잘 모델링되며, 0 근처 작은 그래디언트에 집중되고 강한 엣지가 긴 꼬리를 형성한다. 이 관찰은 BRISQUE \[43\] 등 NSS 기반 NR-IQA의 근간이 되었다. HSMB에서 블록 평균 그래디언트 G̅ 는 분포의 중앙부를 대표하고, 능동 엣지 픽셀(AEP)은 꼬리에 해당한다. 임계값 T = 1.5 × G̅ 로 설정하면 그래디언트 크기 상위 약 75 %가 AEP로 선정되어(Fig. 10), 물리적으로 의미있는 엣지는 보존하고 0 근처 잡음은 배제된다. 1.5 미만에서는 잡음이 AEP에 포함되고, 1.5 초과에서는 약하지만 유용한 엣지가 탈락한다.

**2) JNB (기본값 = 3): Suprathreshold 대비에서의 최소 인지 블러**

Ferzli와 Karam \[65\]이 제안한 Just Noticeable Blur(JNB) 개념은 대비가 Just Noticeable Difference(JND)를 초과하는 조건에서 사람이 인지할 수 있는 최소 엣지 폭 변화를 모델링한다. 표준 시청 조건에서 JNB는 3–5 픽셀 범위로 추정되었다 \[65\]. HSMB에서 JNB 파라미터는 엣지별 블러 확률 P_blur(e_i)가 포화되는 엣지 폭 스케일을 설정한다. 본 연구는 보고된 범위의 하단인 JNB = 3을 채택하여, 엣지 폭이 3 픽셀을 넘는 즉시 인지 가능한 블러로 간주되도록 한다. 이는 MTSS와 같이 경미한 모션블러까지 조기에 감지해야 하는 응용에 적합하다.

**3) β (기본값 = 2): Motion-induced 엣지 폭 분포의 Weibull shape**

Weibull 분포족은 자연 이미지 텍스처·엣지 폭 통계를 기술하는 것으로 확립되었으며 \[76\], shape parameter β는 엣지 폭 분포가 최빈값으로부터 감쇠하는 속도를 결정한다. 특히 β = 2는 Rayleigh 분포에 해당하며, 이는 두 개의 평균-0 Gaussian 성분으로부터 유도되는 반경 크기의 표준 모델 — 즉, Gaussian 광학 PSF가 병진 모션과 컨볼브될 때 엣지 폭 분포가 따르는 형태 — 이다. 따라서 β = 2는 HSMB의 블러 확률 곡선을 motion blur 하에서 기대되는 Rayleigh 형태의 엣지 폭 분포와 정합시킨다. β < 2는 곡선이 sub-Rayleigh로 완만해져 중등도 블러를 과소 벌점하고, β > 2는 곡선이 지나치게 날카로워 중간 블러 수준에서 민감도를 잃는다.

#### E. Ablation Study *(신규, 2026-04-22 — R04 대응)*

세 이론적 기본값을 실험적으로 검증하기 위해, 각 하이퍼파라미터를 기본값에서 독립적으로 sweep 하면서 다른 두 파라미터는 기본값으로 고정한 뒤, Section III의 실험실 HSMB 데이터셋(320장 × 20 조건)에서 HSMB 메트릭과 두 물리적 ground-truth 지표(BEW, MTF50) 간 상관을 측정한다.
