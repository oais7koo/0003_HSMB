# 시나리오: 문서/산출물 생성

> PPT, Word, PDF, 리포트 생성 워크플로우

---

## 개요

마크다운 문서를 PPT, Word, PDF 등 다양한 형식으로 변환하는 워크플로우입니다. 각 스킬이 독립적으로 동작하며, ooreport로 분석 결과를 리포트로 자동 생성할 수도 있습니다.

**이럴 때 사용:** 발표 자료가 필요할 때, 공식 문서를 제출할 때, 분석 결과를 보고서로 만들 때
**결과물:** .pptx / .docx / .pdf 파일


## 1. PPT 프레젠테이션

```bash
ooppt run          # 마크다운 → PPTX 변환
```

**입력:** 마크다운 파일 (제목, 섹션 구조)
**출력:** .pptx 파일

**활용 사례:**
- 프로젝트 발표 자료
- 연구 결과 프레젠테이션
- 제안서

---

## 2. Word 문서

```bash
ooword run         # 마크다운 → DOCX 변환
```

**활용 사례:**
- 견적서 생성
- 공문서 작성
- 보고서

---

## 3. PDF 변환

```bash
oopdf run          # 마크다운 → PDF (weasyprint/pandoc)
```

**지원 엔진:**
- weasyprint: CSS 기반 스타일링
- pandoc: LaTeX 기반 변환

---

## 4. 리포트 자동 생성

```bash
ooreport run       # 분석 결과 → 리포트
```

**파이프라인:**
```
데이터/코드 분석 → 마크다운 리포트 생성 → PPTX/DOCX 변환 (선택)
```

**리포트 유형:**
- 알고리즘 분석 리포트
- 코드 품질 리포트
- 연구 결과 리포트

---

## 5. 조합 활용

| 목적 | 명령 조합 |
|------|----------|
| 연구 리포트 → PPT | `ooreport run` → `ooppt run` |
| PRD → Word 제출 | `ooprd run` → `ooword run` |
| 논문 서머리 → PDF | `oopaper run` → `oopdf run` |
| 코드 분석 → 보고서 | `oocheck run` → `ooreport run` |
