# 시나리오: 도서/유튜브 서머리 생성

> 도서 서머리(7개 섹션) + 유튜브 서머리(8단계) + Word 변환

---

## 개요

oobook 스킬로 도서와 유튜브 영상의 상세 서머리를 자동 생성합니다. 도서는 7개 섹션(장별 요약~퀴즈)으로, 유튜브는 8단계(기초분석~학습자료)로 구성됩니다. 생성된 서머리는 Word 문서로 변환할 수 있습니다.

**이럴 때 사용:** 도서를 체계적으로 정리하고 싶을 때, 유튜브 강의를 텍스트로 정리할 때
**결과물:** data/ps9003/01_도서/*.md (약 50KB/권) + data/ps9003/02_유튜브/*.md

## 전체 흐름

```
Excel 목록 준비 → oobook run → oobook verify → oobook word
```

---

## 방법 1: 도서 서머리 생성

### 1. 입력 데이터 준비

- Excel 파일: `data/ps1001_박기성/BOOKQAI.xlsx`
  - Q1 시트: 도서명 목록 (A열)
  - Q2 시트: 7개 섹션 주제 (A열)
- 샘플 문서: `data/ps1001_박기성/노화도 설계하는 시대가 온다.md`

### 2. 실행

```bash
oobook run                           # 미완성 도서 순차 처리
oobook run --title "깊은 생각의 비밀"   # 특정 도서만
oobook run --batch                   # 전체 배치
```

### 3. 7개 섹션 자동 생성

| 섹션 | 내용 | 분량 |
|------|------|------|
| 1 | 각 장별 핵심내용 정리 | 장별 30줄, 키워드별 3줄 |
| 2 | 핵심 통찰과 울림 구절 30개 | 맥락/의미/원문 + 낭독 스크립트 |
| 3 | 핵심키워드 (30분 서사적 강의용) | 문제제기/원인/해결/변화 |
| 4 | 강의안 작성 | 시간 배분, 서사적 전개 |
| 5 | 강의 전문 (Full Script) | 4,500~5,500자 연설문 |
| 6 | 구글 슬라이드 가이드 | 15-20장 프레젠테이션 |
| 7 | 플래시카드 및 퀴즈 | 카드 20개, 퀴즈 18문항 |

**실행 규칙:**
- 섹션별로 순차 생성 (토큰 제한 방지)
- 섹션 완료 → 파일 저장 → 다음 섹션 자동 진행
- 한 권 완료 → 다음 도서 자동 진행

### 4. 진행 확인

```bash
oobook status    # 전체 진행 상태 (완료/미완료/진행중)
oobook verify    # 완성도 검증
```

---

## 방법 2: 유튜브 서머리 생성

### 1. 입력

- Excel: `data/ps1001_박기성/BOOKQAI.xlsx` (YUTLIST 시트)

### 2. 실행

```bash
oobook youtube              # 유튜브 서머리 생성
```

### 3. 8단계 자동 생성

| 단계 | 내용 |
|------|------|
| 1 | 기초 분석 |
| 2 | 심층 분석 |
| 3 | 핵심 통찰 구절 추출 |
| 4 | 30분 서사적 강의 키워드 설계 |
| 5 | 서사적 강의안 |
| 6 | 강의 전문 |
| 7 | 프레젠테이션 슬라이드 설계 |
| 8 | 학습 보조 자료 |

---

## 방법 3: Word 문서 변환

```bash
oobook word
```

| 항목 | 내용 |
|------|------|
| 입력 | data/ps9003/01_도서/*.md |
| 출력 | data/ps9003/02_워드/*.docx |
| 스타일 | A4, 맑은 고딕/Calibri, 11pt, 1.5줄 간격 |

## 도서 PDF 백업/복원

```bash
oobook backup --dry-run           # 대상 확인
oobook backup                     # 전체 도서 PDF 백업
oobook backup --code BK0200       # 특정 분류만
oobook backup --subdir 01_book_ko # 한국어 도서만
oobook restore --dry-run          # 복원 대상 확인
oobook restore                    # 전체 복원
```

## FAQ

**Q: 한 권에 얼마나 걸리나?**
A: 7개 섹션 순차 생성으로 약 50KB 분량이 생성됩니다. 토큰 제한 때문에 섹션별로 나눠 진행합니다.

**Q: 특정 섹션만 다시 생성할 수 있나?**
A: `oobook run --section 5`로 특정 섹션만 재생성 가능합니다.

**Q: ooscrap의 유튜브 STT와 다른 건가?**
A: ooscrap은 자막 추출(STT→텍스트), oobook youtube는 추출된 텍스트를 체계적으로 분석·강의화하는 것입니다.
