# 빠른 시작 가이드

> OAIS 프로젝트의 주요 시나리오 안내 | `ootutorial run`으로 현행화

---

## 세션 시작/종료

모든 작업의 시작과 끝:

```bash
oostart run        # 세션 시작 (환경 체크, 문서 점검)
oostop run         # 세션 종료 (변경사항 요약, 미완료 체크)
```

## 자주 쓰는 명령 조합

| 목적 | 명령 조합 |
|------|----------|
| 빠른 코드 수정 | `oocheck run` → `oofix run` → `oocommit run` |
| 문서 전체 현행화 | `oodoc run` → `ooenv run` |
| 유튜브 채널 처리 | `ooscrap run` (등록) → `ooscrap run` (STT) |
| 전체 동기화+백업 | `oosync run --push-only` → `oosync backup` |
| 튜토리얼+백업 | `ootutorial run` → `oosync backup` |
