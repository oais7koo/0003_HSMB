# 시나리오: 스크래핑 워크플로우

> URL 등록부터 STT/웹스크래핑, 팩토리 인덱스 갱신까지 전체 흐름

---

## 개요

유튜브 영상과 웹 기사를 자동으로 수집하여 마크다운 텍스트로 보관하는 워크플로우입니다. 유튜브는 자막을 추출(STT)하고, 웹 기사는 본문을 스크래핑합니다. 채널 단위 일괄 처리와 중단 후 재개가 가능합니다.

**이럴 때 사용:** 유튜브 영상을 텍스트로 읽고 싶을 때, 기술 블로그를 옵시디언에 보관할 때
**결과물:** 01_유튜브서머리/*.md + 04_읽을거리/*.md + 유튜브팩토리.md 인덱스


## 전체 흐름

```
URL 등록 (01_다운로드.md) → ooscrap run → 이력 기록 → 팩토리 갱신
```

---

## 방법 1: 통합 배치 처리

### 1. URL 등록
`04_scraping/00_down/01_다운로드.md`의 `## 다운` 섹션에 URL 추가:

```markdown
## 다운
https://youtu.be/xxxxx
https://youtube.com/@채널명
https://blog.example.com/article
```

URL 자동 감지:
| URL 패턴 | 처리 | 출력 |
|----------|------|------|
| `youtu.be/*`, `youtube.com/watch*` | STT (자막→Whisper) | `01_유튜브서머리/` |
| `youtube.com/@*` | 채널 스캔→등록→STT | `01_유튜브서머리/` |
| `youtube.com/shorts/*` | 스킵 (자동 삭제) | - |
| 그 외 URL | 웹 스크래핑 | `04_읽을거리/` |

### 2. 실행
```bash
oocontext 4        # SP04 전환
ooscrap run        # 전체 처리
```

### 3. 결과 확인
- 성공 URL → 01_다운로드.md에서 자동 제거
- 실패 URL → `# [FAIL] URL | 사유` 주석 처리
- 이력 → 02_유튜브동영상리스트.md / 04_읽을거리리스트.md

---

## 방법 2: 채널 전체 처리 (등록→처리 2단계)

### 1. 채널 URL 등록
```markdown
## 다운
https://youtube.com/@happyai_0612
```

### 2. 1차 실행 (스캔+등록)
```bash
ooscrap run
```
- 채널 영상 목록 추출 (yt-dlp)
- 쇼츠(60초 미만) 자동 제외
- 이력 대비 중복 제거
- 신규 영상 URL을 01_다운로드.md에 등록
- 03_유튜브채널리스트.md에 채널 메타 기록

### 3. 2차 실행 (STT 처리)
```bash
ooscrap run                    # 전체 처리
ooscrap summary --subtitle-only  # 자막만 빠르게 (Whisper 없이)
```

### 4. 채널 재스캔 (신규 영상 추가)
채널 URL을 다시 01_다운로드.md에 넣으면 신규 영상만 추가 등록.

---

## 방법 3: 자막 전용 모드

```bash
ooscrap summary --subtitle-only   # 자막 있는 것만 (건당 ~5초)
```
- 자막 없는 영상은 FAIL 처리
- 나중에 Whisper로 재처리 가능

## 이력 파일 구조

| 파일 | 역할 |
|------|------|
| `00_down/01_다운로드.md` | 처리할 URL (처리 후 자동 제거) |
| `00_down/02_유튜브동영상리스트.md` | 동영상 처리 이력 (중복 체크 기준) |
| `00_down/03_유튜브채널리스트.md` | 채널 메타 (채널명/스캔일/영상수) |
| `00_down/04_읽을거리리스트.md` | 웹 기사 처리 이력 |

## FAQ

**Q: naver 블로그/aitimes가 안 되는데?**
A: WebFetch에서 접근 차단. 브라우저에서 직접 복사 필요.

**Q: 채널 처리 중 중단하면?**
A: 미처리 URL이 01_다운로드.md에 남아서 재실행하면 이어서 처리.

**Q: 동영상/음악 다운로드는?**
A: `## 동영상`, `## 음악` 섹션에 URL 추가 후 ooscrap run.
