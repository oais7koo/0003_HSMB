# ooppt Tutorial

> PPT 노트 생성 스킬. 대상 MD 문서를 PPT 발표용 슬라이드 노트 형식의 MD 문서로 변환한다. | 버전: v05 | 카테고리: content

## 1. 이 스킬은 왜 필요한가?

PPT 노트 생성 스킬. 대상 MD 문서를 PPT 발표용 슬라이드 노트 형식의 MD 문서로 변환한다.

## 2. 빠른 시작 (5분 가이드)

```bash
# 기본 실행
ooppt run

# 상태 확인
ooppt status

# 도움말
ooppt help
```

## 3. 전체 서브명령어

| 명령어 | 설명 |
|--------|------|
| `ooppt help` | 서브명령어 목록 표시 |
| `ooppt version` | 스킬 버전 정보 (v05) |
| `ooppt run [경로]` | **MD → PPT노트 MD 변환 실행** |
| `ooppt update [경로]` | 현행화 — 소스 변경분 감지 → PPT 재생성 | .pptx |
| `ooppt update --dry-run` | 변경 예정 내용 미리 출력 (실제 수정 안 함) | 터미널 |
| **`ooppt run this`** | **직전 작업 PPT 노트 변환** (→ common_guide.md §9) |
| `ooppt run --slides N` | 슬라이드 수 지정 (기본: 자동) |
| `ooppt run --style [비즈니스\|기술\|교육]` | 발표 스타일 지정 |

## 4. 상세 사용법

### 스킬 요약

| 항목 | 내용 |
|------|------|
| **핵심 역할** | 대상 MD 문서를 PPT 발표용 슬라이드 노트 형식 MD로 변환 |
| **하는 것** | MD→슬라이드 노트 MD 변환, 발표 분량 조절, 섹션별 노트 구성 |
| **하지 않는 것** | PPTX 파일 직접 생성(→document-skills:pptx), Word 변환(→ooword), 리포트 생성(→ooreport) |
| **참조 범위** | 현재 프로젝트 내부 파일만 / 외부 프레젠테이션 서비스 자동 연동 안 함 |
| **수정 대상** | 입력 MD 기반 새 `*_노트.md` 파일 생성 |
| **실행 레벨** | [반자동] — 변환 구조 확인 후 생성 |
| **에이전트 호환** | 범용 — 파일 읽기·쓰기 작업 중심으로 모든 에이전트 처리 가능 |

### 출력 형식

```markdown
# {제목} - PPT 노트

> 원본: {입력파일 경로} | 생성: {날짜} | 슬라이드: {N}장

---

### 슬라이드 1. 표지

챕터:
제목: {문서 제목}
부제: {핵심 메시지 1줄}

---

### 슬라이드 2. 개요

챕터: 01 Overview
제목: 발표 목차
부제:

□ 첫 번째 주제
□ 두 번째 주제
□ 세 번째 주제

---

### 슬라이드 3. {섹션 제목}

챕터: 02 {영문 키워드}
제목: {슬라이드 제목}
부제: {보조 설명}

□ {핵심 포인트 1}
  ○ {세부 내용}
  ○ {세부 내용}

□ {핵심 포인트 2}
  ○ {세부 내용}

---
```

### 출력 규칙

| 항목 | 규칙 |
|------|------|
| **출력 위치** | 입력 파일과 동일한 디렉토리 |
| **파일명** | `{원본명}_ppt노트.md` |
| **슬라이드 수** | 10~15장 (자동), `--slides N`으로 조정 |
| **환각 방지** | 원본에 없는 내용 추가 금지 |

**예시**:
- 입력: `04_기술연구/42_제품핵심기술.md`
- 출력: `04_기술연구/42_제품핵심기술_ppt노트.md`

## 5. 워크플로우

```
입력 MD 읽기 → 핵심 내용 추출 → 슬라이드 구조 설계 → PPT노트 MD 생성
```

### 3.1 핵심 내용 추출

- 제목/섹션 계층 분석
- 핵심 메시지 식별 (□/■ 마커, 굵은 텍스트, 결론 문장)
- 데이터/수치 추출 (표, 통계, 비교)
- 불필요 내용 제거 (상세 코드, 긴 설명, 참조 링크)

### 3.2 슬라이드 구조 설계

| 슬라이드 유형 | 용도 | 콘텐츠 |
|-------------|------|--------|
| 표지 | 제목, 부제, 발표자 | 문서 제목 + 요약 1줄 |
| 개요/목차 | 발표 흐름 | 주요 섹션 3~5개 |
| 내용 | 핵심 메시지 전달 | 제목 + 불릿 3~5개 |
| 비교 | 대조/분석 | 2~3칼럼 비교표 |
| 데이터 | 수치/통계 | KPI, 차트용 데이터 |
| 요약 | 마무리 | 핵심 메시지 3개 |

### 3.3 슬라이드당 콘텐츠 규칙

| 항목 | 규칙 |
|------|------|
| 제목 | 1줄, 10단어 이내 |
| 불릿 | 3~5개, 각 2줄 이내 |
| 키워드 | 문장이 아닌 핵심 키워드 중심 |
| 데이터 | 수치는 반드시 포함 |
| 원문 유지 | 환각 금지, 원본 내용만 사용 |
