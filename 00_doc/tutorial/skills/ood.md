# ood Tutorial

> oodev 스킬의 약어 alias — TDD 기반 개발 | 버전: v03 | 카테고리: core-dev

## 1. 이 스킬은 왜 필요한가?

`ood` = `oodev` 단축 호출. TDD 워크플로우(RED → GREEN → REFACTOR)로 기능을 개발하고, 구현 단계를 자동으로 전환합니다.

## 2. 빠른 시작 (5분 가이드)

```bash
# 기본 실행
ood run

# 상태 확인
ood status

# 도움말
ood help
```

## 3. 자주 쓰는 명령 요약

| # | 명령어 | 언제 쓰나 |
|---|--------|----------|
| 1 | `ood run` | 상세 문서 기반 TDD 개발 전체 실행 |
| 2 | `ood run dXXXX` | 특정 상세 문서 개발 |
| 3 | `ood status` | 개발 진행 상황 확인 |
| 4 | `ood help` | 서브명령어 도움말 |
