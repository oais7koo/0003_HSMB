# ook Tutorial

> oocheck 스킬의 약어 alias — 코드 품질 체크 | 버전: v03 | 카테고리: core-dev

## 1. 이 스킬은 왜 필요한가?

`ook` = `oocheck` 단축 호출. Python 코드 품질을 pylint/mypy/pytest로 검증하고, 발견된 에러를 자동 분류하여 d0004_todo.md에 등록합니다.

## 2. 빠른 시작 (5분 가이드)

```bash
# 기본 실행
ook run

# 상태 확인
ook status

# 도움말
ook help
```

## 3. 자주 쓰는 명령 요약

| # | 명령어 | 언제 쓰나 |
|---|--------|----------|
| 1 | `ook run` | 코드 전체 품질 검사 실행 |
| 2 | `ook run dXXXX` | 특정 상세 개발 문서 검증 |
| 3 | `ook status` | 검증 진행 상황 확인 |
| 4 | `ook help` | 서브명령어 도움말 |
