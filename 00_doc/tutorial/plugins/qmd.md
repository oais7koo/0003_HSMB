# MCP 플러그인: qmd

## 개요

`qmd` MCP 서버. 프로젝트 `.mcp.json`에 등록되어 자동 연결됩니다.

## 설정

| 항목 | 값 |
|------|-----|
| command | `qmd.cmd` |
| args | `mcp --collection 1_oais` |

## 제공 도구

(MCP 도구는 세션 연결 시 자동 로드)

## 사용 예시

`.mcp.json`에 등록 후 자동 사용 가능

## 활용 시나리오

- `--seq` 플래그로 활성화
- 복잡한 분석/디버깅 시 자동 활성화
