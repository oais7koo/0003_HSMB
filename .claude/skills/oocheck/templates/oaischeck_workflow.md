# 병렬 실행 워크플로우 템플릿

> 중앙 안내: 상세한 에이전트 활용 원칙 및 공통 가이드라인은 `.claude/guides/common_guide.md`의 '2. 에이전트 활용 원칙' 섹션을 참조하십시오.

## 템플릿 설명

이 템플릿은 병렬 에이전트 기반 워크플로우를 시각화하기 위한 다이어그램입니다.
프로젝트별로 에이전트 이름과 단계를 커스터마이즈하여 사용하세요.

## 기본 다이어그램 템플릿

```
┌─────────────────────────────────────────────────────────────┐
│                    [메인 명령어] 실행                        │
└─────────────────────────────────────────────────────────────┘
                           │
                           ▼
    ┌──────────────────────┬──────────────────────┐
    │                      │                      │
    ▼                      ▼                      ▼
┌────────────┐      ┌────────────┐      ┌────────────┐
│ [에이전트1] │      │ [에이전트2] │      │ [에이전트3] │
│            │      │             │      │            │
│ (병렬 1/3) │      │ (병렬 2/3) │      │ (병렬 3/3) │
└────────────┘      └────────────┘      └────────────┘
    │                      │                      │
    └──────────────────────┼──────────────────────┘
                           │
                           ▼
              ┌────────────────────────┐
              │ 결과 통합 및 우선순위  │
              │ P1: 즉시 수정 필요     │
              │ P2: 권장 수정          │
              │ P3: 개선 권장          │
              └────────────────────────┘
                           │
                           ▼
              ┌────────────────────────┐
              │ [결과 기록 문서] 기록  │
              └────────────────────────┘
```
