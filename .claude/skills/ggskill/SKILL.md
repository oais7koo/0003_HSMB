## 목적

- oo스킬(클로드코드 전용)을 Copilot 환경에서 동작 가능한 스킬로 자동 변환
- SKILL.md, scripts/ 구조를 분석하여 Copilot에서 불가능한 부분(모델, 플러그인 등) 자동 감지 및 대체/삭제
- 변환 결과를 새로운 gg스킬 폴더에 생성

## 사용법

1. 변환 대상 oo스킬 폴더 지정
2. copilot에서 동작 가능한 명령어/흐름만 추출
3. 변환 결과를 gg스킬 폴더에 생성

## 제한사항

- Copilot에서 지원하지 않는 모델/플러그인/CLI는 자동으로 제거 또는 대체
- 복잡한 워크플로우는 최소 기능만 변환

## 예시

- ootodo → ggtodo 변환

## 명령어

- `convert.py <oo스킬폴더> <gg스킬폴더>`
