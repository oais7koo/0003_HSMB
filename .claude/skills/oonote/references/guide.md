## 1. 개요

**oonote**: 연구노트(d0011) 작성·관리.

- **참조**: SKILL.md (서브명령어·워크플로우)
- **이 문서**: 방법론(How) — 실행 패턴, 입력/출력 형식, 사용 가이드

---

## 2. 기본 사용법

실행: `uv run python .claude/skills/oonote/scripts/oonote_run.py [args]`

---

## 3. 워크플로우

### 3.1 `oonote add "내용"` 흐름

```
1. d{SP}0020_연구노트.md 존재 확인 (없으면 자동 생성)
2. 현재 날짜(YYYY-MM-DD) 섹션 확인
   - 오늘 날짜 섹션 없음 → 새 섹션 생성
   - 오늘 날짜 섹션 있음 → 해당 섹션 끝에 추가
3. 현재 시간(HH:MM) + 내용 기록
4. 문서이력관리 버전 자동 증가
```

### 3.2 노트 구조

```markdown
## 2026-03-21 (토)

### 10:30 | [태그] 제목
내용 본문

**참조**: 관련 파일 또는 링크 (있을 경우)

---
```

## 4. 입출력 형식

| 항목 | 내용 |
|------|------|
| 입력 | 서브명령어 인자 또는 현재 SP 컨텍스트 |
| 출력 | 터미널 출력 또는 문서 파일 생성 |
| 로그 | 에러 발생 시 d{SP}0004_todo.md 등록 |

### 4.1 연구노트 문서 형식

```markdown
# [프로젝트명] 연구노트

## 문서이력관리
- v01 2026-03-21 — 초기 생성

---

## 2026-03-21 (토)

### 10:30 | [실험] 모델 A 결과 확인
- Loss: 0.234, Accuracy: 91.2%
- 이전 대비 1.8% 향상
- 다음: 배치 크기 조정 실험 필요

**참조**: `00_doc/sp00/d0002_plan.md`

---

### 14:00 | [회의] 주간 미팅 메모
- ...

---
```

### 4.2 `oonote search` 출력 예시

```
검색: "Loss"  |  3건 발견

[2026-03-21 10:30] [실험] 모델 A 결과 확인
  Loss: 0.234, Accuracy: 91.2%

[2026-03-20 15:00] [실험] 베이스라인 측정
  Loss: 0.252, ...
```

### 4.3 `oonote status` 출력 예시

```
[oonote status]
파일  : 00_doc/sp00/d0020_연구노트.md
전체  : 47개 노트 | 12일 기록
이번달: 8개 노트 (2026-03)
오늘  : 2개 노트 (2026-03-21)
태그  : [실험] 18, [분석] 12, [회의] 7, ...
```
