## 1. 에러 개요

### 1.1 에러 정보

| 항목 | 내용 |
|------|------|
| **에러 유형** | [SyntaxError / TypeError / ValueError / ...] |
| **우선순위** | [P1-즉시 / P2-권장 / P3-개선] |
| **발견 경로** | [oocheck / 수동 / 테스트 / 사용자 보고] |
| **발견 일시** | YYYY-MM-DD HH:MM |

### 1.2 에러 메시지

```text
에러 메시지 전문을 여기에 붙여넣기
```text

### 1.3 발생 위치

- **파일**: `파일 경로`
- **라인**: 라인 번호
- **함수/클래스**: 함수명 또는 클래스명
