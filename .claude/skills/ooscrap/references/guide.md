## 3. 워크플로우

### 3.1 통합 처리 프로세스 (ooscrap run)

```
1. 01_다운로드.md의 ## 다운 섹션에서 URL 읽기
2. URL 패턴으로 자동 분류 (유튜브 동영상 / 유튜브 채널 / 일반 URL)
3. 이력 파일 로드 (12_동영상리스트 + 14_읽을거리리스트) → 중복 URL 스킵 (--force 시 무시)

--- Phase 1: 채널 이외 URL 먼저 처리 (즉시 삭제) ---

4. [일반 URL] 각각 {
  4-1) WebFetch로 웹페이지 전문 가져오기
  4-2) 본문 텍스트 추출 → 마크다운 변환
  4-3) 04_읽을거리/YYMMDD-HHMMSS_첫5단어.md 저장
  4-4) 04_읽을거리리스트.md에 이력 기록
  4-5) ★ 성공/스킵 시 01_다운로드.md에서 해당 URL 즉시 제거, 실패 시 # [FAIL] 주석
}

5. [유튜브 동영상 URL] 각각 {
  5-1) 영상 정보 추출 (제목, 자막 유무)
  5-2) 자막 우선: 수동자막(ko) > 자동자막(ko) > Whisper STT 폴백
  5-3) 01_유튜브서머리/YYMMDD-HHMMSS_첫5단어.md 저장
  5-4) 12_유튜브동영상리스트.md에 이력 기록
  5-5) ★ 성공/스킵 시 01_다운로드.md에서 해당 URL 즉시 제거, 실패 시 # [FAIL] 주석
}

6. [## 동영상 섹션] URL별 반복 (중복 체크 없음) {
  6-1) yt-dlp로 mp4/webm 다운로드
  6-2) 02_유튜브동영상/YYMMDD-HHMMSS_첫5단어.mp4 저장
  6-3) ★ 성공 시 01_다운로드.md에서 즉시 제거
}

7. [## 음악 섹션] URL별 반복 (중복 체크 없음) {
  7-1) yt-dlp로 오디오 추출 → mp3 변환
  7-2) 03_유튜브음악/YYMMDD-HHMMSS_첫5단어.mp3 저장
  7-3) ★ 성공 시 01_다운로드.md에서 즉시 제거
}

--- Phase 2: 채널 URL → 영상 목록 전개 후 ## 다운에 추가 ---

8. [유튜브 채널 URL] 각각 {
  8-1) 13_유튜브채널리스트.md에서 중복 체크 → 신규면 등록
  8-2) yt-dlp로 채널 전체 영상 목록 추출 (URL 목록만, 다운로드 X)
  8-3) 쇼츠 제외 (60초 미만 영상 필터링)
  8-4) 12_유튜브동영상리스트.md에서 중복 체크 → 미처리 영상 URL만 필터링
  8-5) ★ 미처리 영상 URL을 01_다운로드.md의 ## 다운 섹션 맨 앞에 일괄 추가
  8-6) ★ 채널 URL을 01_다운로드.md에서 즉시 제거
  8-7) 13_유튜브채널리스트.md 업데이트 (마지막처리일, 추가된 영상수)
}
  ※ 채널에서 전개된 영상 URL은 이후 Phase 1 Step 5 흐름으로 처리됨
  ※ "전체 처리"가 아니라 "목록 등록 → 개별 처리"로 분리

9. Phase 2 완료 후 새로 추가된 영상 URL을 Phase 1 Step 5 흐름으로 순차 처리
   (각 영상 처리 후 01_다운로드.md에서 해당 줄 즉시 제거)

10. 처리 결과 요약 출력 (서머리 N건, 채널 전개 N건/N개 영상, 읽을거리 N건, 동영상 N건, 음악 N건)
```

### 3.2 read 서브명령어 프로세스

```
1. 01_다운로드.md의 ## 읽을거리 섹션에서 URL 목록 읽기
2. 02_유튜브동영상리스트.md에서 기존 URL 로드 (중복 방지, --force 시 스킵)
3. URL별 반복 {
  3-1) WebFetch로 웹페이지 전문 가져오기
  3-2) 본문 텍스트 추출 → 마크다운 변환
  3-3) 04_읽을거리/YYMMDD-HHMMSS_첫5단어.md 저장
  3-4) 02_유튜브동영상리스트.md에 이력 기록
}
4. 01_다운로드.md 정리 (성공 제거, 실패 → # [FAIL] URL | 사유)
5. 처리 결과 요약 출력
```

## 4. 입출력 형식

### 4.1 01_다운로드.md 입력 형식

```markdown
## 다운
https://youtu.be/xxxxx
https://youtube.com/@channel
https://blog.example.com/article

## 동영상
https://youtu.be/yyyyy

## 음악
https://youtu.be/zzzzz
```

### 4.2 유튜브 서머리 출력 형식

```markdown
# 동영상 제목

> URL: https://youtu.be/xxx
> 날짜: 2026-02-19 14:30
> 소스: 자막 (자동)

## 트랜스크립트

| 시간 | 내용 |
|------|------|
| 00:00:00 - 00:00:05 | 안녕하세요 오늘은... |
| 00:00:05 - 00:00:12 | 이 영상에서는... |
```

### 4.3 읽을거리 출력 형식

```markdown
# 기사/블로그 제목

> URL: https://blog.example.com/article
> 날짜: 2026-03-22 14:30
> 소스: 웹 스크래핑

## 본문

(전문 마크다운)
```
