# 기술수요조사 체크리스트

> 공통 규칙 상속: `../checklist_common.md` (G01~G18)
> 이 문서는 기술수요조사 프로세스에 특화된 추가 규칙만 정의

## step.0 논문 수집

| # | 규칙 | 검증 방법 |
|---|------|---------|
| P01 | 수집 논문 폴더 생성 | 03_paper/11_paper_en/{폴더ID}/ 존재 |
| P02 | 서머리 완료 | *_00_*서머리.md 파일 존재 |
| P03 | paper_list 반영 | paper_list.md에 수집 논문 등재 |

## step.1 서베이

| # | 규칙 | 검증 방법 |
|---|------|---------|
| S01 | 필수 섹션 포함 | 개요, 기술 동향, 시사점, 참고문헌 |
| S02 | 기술 갭 분석 포함 | 추가 연구 필요 영역 섹션 존재 |
| S03 | 인용 폴더ID 표기 | 인용 논문에 [폴더ID] 명시 |

## step.2 기술연구

| # | 규칙 | 검증 방법 |
|---|------|---------|
| R01 | 비교표 폴더ID 컬럼 | 기술 비교 테이블에 폴더ID 컬럼 포함 |
| R02 | 범위 외 요건 배제 | 타 구성기술 범위(물리 요건 등) 미포함, 참고 노트로만 언급 |
| R03 | 서베이 참조 링크 | 서베이 파일 경로 명시 |

## step.3 기술수요

| # | 규칙 | 검증 방법 |
|---|------|---------|
| D01 | 논문 인용 0건 | `[26` 패턴, `et al.` 패턴 grep 0건 |
| D02 | 기호 0건 | `·` `→` `/` `~` grep 0건 (센서명 구분 쉼표 제외) |
| D03 | 구체 기술명 일반화 | 특정 모델명 → 일반 서술 |
| D04 | 헤더 형식 | `# 기술수요조사서 — {구성기술}` |
| D05 | 기술연구 참조 링크 | 하단에 참조 1줄만 |

| # | 규칙 | 검증 방법 |
