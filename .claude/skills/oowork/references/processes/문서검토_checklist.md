# 문서검토 체크리스트

> 공통 규칙 상속: `../checklist_common.md` (G01~G18)
> 이 문서는 문서검토 프로세스에 특화된 추가 규칙만 정의

## step.0 검토기준 설정

| # | 규칙 | 검증 방법 |
|---|------|---------|
| R01 | 검토기준 파일 존재 | 00_검토기준/01_검토기준.md 존재 |
| R02 | 항목 ID 부여 | 모든 항목에 U01~ ID 부여 |
| R03 | 심각도 지정 | 각 항목에 Critical/Major/Minor 심각도 명시 |

## step.1 문서 추출

| # | 규칙 | 검증 방법 |
|---|------|---------|
| R04 | 원본 보존 | 01_원본/에 원본 파일 존재 |
| R05 | 추출 파일 생성 | 02_추출/ 내 .md 파일 1개 이상 |
| R06 | 변환 품질 | 표·목차·특수문자가 원본과 대응 |

## step.2 구조 분석

| # | 규칙 | 검증 방법 |
|---|------|---------|
| R07 | 목차 추출 | 섹션 목록이 원본 목차와 일치 |
| R08 | 분량 분석 | 섹션별 분량(글자수/비중) 기재 |
| R09 | 문서 유형 판별 | 문서 유형(제안서/보고서 등) 명시 |

## step.3 내용 검토

| # | 규칙 | 검증 방법 |
|---|------|---------|
| R10 | 심각도 분류 | 모든 지적사항에 Critical/Major/Minor 분류 |
| R11 | 근거 명시 | 지적사항마다 해당 위치(섹션/페이지) 명시 |
| R12 | 5대 관점 검토 | 논리·정확성·일관성·완성도·가독성 전수 검토 |
| R13 | 사용자 기준 전수 검토 | U01~ 항목 각각에 대해 판정 결과(적합/부적합/해당없음) 기재 |
| R14 | 통합 결과표 | 기본 관점 + 사용자 기준이 하나의 테이블로 통합 |

## step.4 수정 제안

| # | 규칙 | 검증 방법 |
|---|------|---------|
| R15 | 검토-수정 매핑 | 검토 지적사항 전수에 대해 수정안 제시 |
| R16 | 현재-수정 대비 | 각 항목에 현재 표현과 수정안 병기 |
| R17 | 우선순위 분류 | 즉시/단기/선택 우선순위 분류 |
