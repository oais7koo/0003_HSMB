
## 1. 큰제목
본 연구는 터널 영상 데이터셋의 신뢰성을 높이기 위하여, 웹 기반 플랫폼 상에서 자동화된 품질 평가와 보정 기법을 통합하는 방안을 제시한다. 플랫폼은 대용량 영상을 청크 전송으로 수집하며 전송 과정에서 무결성을 검증한다. 수집된 영상은 이미지 품질 평가 모듈을 통해 짧은 시간 안에 SSIM, PSNR, BRISQUE, CPBD와 같은 지표가 산출되고, 여러 지표를 가중합하여 일관된 품질 등급이 부여된다.

등급이 기준 이하로 판정되면 모션블러 복원, 노이즈 억제, 조도 균일화 등이 자동 실행되어 선명도와 변별력이 눈에 띄게 향상된다. 이러한 전 과정을 하나의 웹 애플리케이션에서 처리함으로써 현장 요구 수준의 실시간 분석과 즉시 리포트 생성을 지원하고, 권한 관리 및 로그 보존 체계를 통해 보안과 감사 요건도 충족한다.

결과적으로 데이터 정제 부담과 딥러닝 모델의 재학습 필요성이 완화되어 유지관리 의사결정이 신속해지며, 향후 최신 NR-IQA 알고리즘 추가 및 현장 촬영 장비 연동을 통해 적용 영역을 확장할 수 있는 잠재력을 지닌다.

연구 개발 범위는 다음과 같다. 
- 풀참조방식 이미지품질 측정(Full Reference Image Quality Assessment(FR-IQA)) 기능 3종 개발
-  비참조방식 이미지품질 측정(No-reference Image Quality assessment (NR-IQA)) 기능 3종 개발
- 터널영상 데이터셋 품질 측정 웹애플리케이션 파일럿 개발

### 가. 작은제목
각 데이터셋은 터널 표면의 영상을 촬영한 후 처리에 적합한 크기로 분할한 이미지로 구성되며, 대상 시설의 종류를 클래스로 하고 내부 이미지 별로 IQA 측정값과 MOS를 가지고 있다. 또한 각 데이터셋은 포함 이미지의 컬러분포, 밝기, 휘도, 표면 패턴 분포 등의 특징벡터(feature vector)정보를 메타정보로 가지고 있음
IQA는 NR-IQA인 BRISQUE, PIQE, NIQE, CPBD 등을 대상으로 한다.
MOS는 터널 표면 영상의 선명도를 복수의 관측자가 0~1의 범위를 0.1 단위로 선명도가 높을 수록 1이 되도록 분류하도록 하고 각 영상의 선명도는 이를 평균한 MOS(Mean Opinion Score)로 정의한다.

### 나. 작은제목목
각 데이터셋 별로 이미지의 특징벡터, IQA 벡터와 MOS간의 관계를 회귀분석, Decision tree 등의 머신 러닝 알고리즘을 사용해서 이미지의 특징벡터, IQA 벡터를 입력데이터로 하고, MOS를 목표데이터로 하는 모델들을 기준 데이터셋 별로 구축한다.
각 데이터셋-모델은 학습데이터의 종류에 따라서 클래스를 갖는다. 예를 들어서 상위 클래스는 터널, 교량 등의 시설 종류이고 하위 클래스는 누수, 균열, 박리 등 검침을 하고자 하는 결함 종류이다.
이미지 품질 측정에서는 입력 이미지 데이터셋의 상-하위 클래스에 해당되는 데이터셋 클래스의 모델만 대상으로 해서 특징(feature) 정보와 각 데이터셋의 메타정보를 바탕으로 Softmax 가중 앙상블 모델을 적용한다.
데이터셋 클래스는 터널-균열, 터널-박리, 터널-누수 등 대상시설물과 검지하고자 하는 결함의 조합으로 이루어진다.
입력이미지의 feature 가 각 데이터셋의 feature 분포에서 해당되는 밀도의 강도에 따라서 전체 합이 1이 되도록 하는 가중치를 데이터셋 별로 부여하고, 각 데이터셋 별 모델에서 MOS를 산출한 후 각각의 MOS에 이 가중치를 곱해서 다한 값을 입력이미지의 MOS로 도출한다.

## 2 큰제목
본 플랫폼은 터널 영상 데이터 수집-검증-보정-리포트까지 전 과정을 단일 파이프라인으로 연결함으로써, 현장 촬영 직후 품질 저하 영상을 즉시 식별하고 재촬영 또는 자동 복원을 유도한다. 이를 통해 관리자는 데이터 정제와 모델 재학습 부담을 크게 줄이면서도 균열 탐지 정확도를 안정적으로 확보할 수 있고, 유지보수 의사결정은 신속-선제형으로 전환된다. 또한 품질 기준과 절차가 시스템에 내장돼 조직 간 경험 편차가 완화되고, 플랫폼 구조가 모듈화돼 교량·옹벽 등 타 구조물 영상에도 확장할 수 있다.

산업계에서는 장비-네트워크-AI 모델을 통합한 디지털 전환 기반으로 활용될 수 있으며, 학계에서는 도메인 특화 IQA 연구와 데이터 기반 안전관리 지침 마련에 실증 데이터를 제공할 수 있다. 실시간 통계 누적을 통해 노후화 추세를 분석하고 예방 보수를 계획할 수 있으며, 공공 발주 기준 마련과 민간 서비스 고도화에도 기여해 산업 생태계 경쟁력을 높이는 촉매 역할을 할 것으로 기대된다.
