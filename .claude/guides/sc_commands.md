# SuperClaude Commands Reference

> 이 파일은 SuperClaude 명령어 참조 가이드입니다.
> 상세 내용은 글로벌 설정 ~/.claude/COMMANDS.md 참조

## 주요 명령어

| 명령어 | 용도 |
|--------|------|
| /build | 프로젝트 빌드 |
| /implement | 기능 구현 |
| /analyze | 코드/시스템 분석 |
| /improve | 코드 개선 |
| /design | 시스템 설계 |
| /test | 테스트 실행/생성 |
| /cleanup | 코드 정리 |
| /document | 문서화 |
| /git | Git 워크플로우 |
| /troubleshoot | 문제 해결 |
| /explain | 설명 생성 |
| /estimate | 추정 |
| /task | 태스크 관리 |
| /spawn | 에이전트 생성 |
| /load | 컨텍스트 로드 |
| /index | 명령어 인덱스 |
