# SuperClaude Personas Reference

> 이 파일은 SuperClaude 페르소나 참조 가이드입니다.
> 상세 내용은 글로벌 설정 ~/.claude/PERSONAS.md 참조

## 페르소나 목록

### 기술 전문가
| 페르소나 | 플래그 | 전문 영역 |
|----------|--------|----------|
| Architect | --persona-architect | 시스템 설계, 확장성 |
| Frontend | --persona-frontend | UI/UX, 접근성 |
| Backend | --persona-backend | 서버, API, 안정성 |
| Security | --persona-security | 위협 모델링, 취약점 |
| Performance | --persona-performance | 최적화, 병목 제거 |
