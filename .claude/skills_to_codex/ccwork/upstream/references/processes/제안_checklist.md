## step.0 RFP 분석

| # | 규칙 | 검증 방법 |
|---|------|---------|
| A01 | 문서 식별 | 문서명, 발주처, 유형(RFP/양식/공고) 기재 |
| A02 | 원본 문서 목록 | 11_RFP분석.md 상단에 원본 목록 테이블 존재 — 파일명, 유형, 페이지수, 분석여부(O/X), 미분석 0건 |
| A03 | 핵심 요구사항 추출 | 사업 개요, 목적, 범위, 일정, 예산 중 해당 항목 기재 |
| A04 | 양식 항목 분해 | 양식/RFP의 작성 항목이 빠짐없이 나열됨 |
| A05 | 제출 서류 목록 | 필수 제출 서류가 나열됨 — 해당 시 |
| A06 | 제약 조건 정리 | 기간/인력/기술/보안 제약이 명시됨 — 해당 시 |
| A07 | 양식분석 문서 | 12_양식분석.md 존재 — 필드목록, 입력형식, 필수/선택 구분 포함 |
