# oopdf check 체크리스트

> oopdf 스킬 역할 수행 품질 검증 항목 (PDF 변환 결과 품질)

| ID | 항목 | 검증 내용 | 심각도 |
|----|------|----------|--------|
| C01 | 레이아웃 가독성 | 변환된 PDF에서 텍스트 잘림, 단락 겹침, 여백 비정상 등 레이아웃 깨짐이 없는지 확인 | CRITICAL |
| C02 | 한글 폰트 렌더링 | 한글 문자가 깨지거나 박스(□)로 표시되지 않고 정상 렌더링되었는지 확인 | CRITICAL |
| C03 | 표 변환 정확성 | Markdown 표가 PDF에서 테두리·열·행 정렬이 유지되며 데이터 손실 없이 표시되는지 확인 | ERROR |
| C04 | 코드 블록 변환 정확성 | 코드 블록이 고정폭 폰트로 들여쓰기와 줄바꿈이 보존된 상태로 표시되는지 확인 | ERROR |
| C05 | 수식 렌더링 (--pandoc) | `--pandoc` 옵션 사용 시 LaTeX 수식이 깨지지 않고 수식 이미지 또는 텍스트로 정상 렌더링되었는지 확인 | WARNING |
| C06 | 출력 경로 규칙 준수 | `--output` 미지정 시 입력 파일과 동일한 폴더에 동일한 파일명(확장자만 `.pdf`)으로 생성되었는지 확인 | WARNING |
| C07 | 제목 계층 구조 유지 | H1~H4 제목이 PDF에서 크기·굵기 차이로 계층이 시각적으로 구분되는지 확인 | INFO |
| C08 | 이미지 변환 품질 (--image) | `convert --image` 시 모든 페이지가 지정 DPI·포맷으로 변환되고 텍스트 가독성이 유지되는지 확인 | ERROR |
| C09 | 이미지 출력 경로 규칙 (convert) | `--output` 미지정 시 입력 PDF와 동일 폴더에 `{파일명}/` 디렉토리 생성 후 `{파일명}_01.png` (100p 미만) 또는 `{파일명}_001.png` (100p 이상) 형식으로 저장되는지 확인 | WARNING |

## check 출력 형식

```
[oopdf check]
C01 레이아웃 가독성          [OK]
C02 한글 폰트 렌더링         [OK]
C03 표 변환 정확성           [OK]
C04 코드 블록 변환 정확성    [OK]
C05 수식 렌더링 (--pandoc)   [WARN]
C06 출력 경로 규칙 준수      [OK]
C07 제목 계층 구조 유지      [OK]
소계: OK:6 | WARN:1 | ERROR:0
```

## 심각도 기준

| 심각도 | 기호 | 의미 |
|--------|------|------|
| CRITICAL | [ERROR] | 스킬 역할 완전 실패 |
| ERROR | [ERROR] | 핵심 기능 오류 |
| WARNING | [WARN] | 품질 저하 |
| INFO | [INFO] | 개선 권장 |
