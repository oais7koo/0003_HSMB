# Data Analysis Command: 데이터 분석 및 시각화

## 명령어 목적
Python 데이터 사이언스 프로젝트에서 데이터 분석, 전처리, 시각화 작업을 효율적으로 수행합니다.

## 작업 프로세스

### 1. 데이터 탐색 단계
- **데이터 로딩**: pandas, numpy를 활용한 데이터 읽기
- **기본 정보 확인**: shape, dtypes, missing values, 기술통계
- **데이터 품질 평가**: 중복값, 이상치, 데이터 분포 확인

### 2. 전처리 단계
- **결측치 처리**: 삭제, 대체, 보간 방법 선택
- **이상치 처리**: IQR, Z-score 기반 탐지 및 처리
- **데이터 변환**: 정규화, 표준화, 로그 변환
- **피처 엔지니어링**: 새로운 변수 생성, 범주형 인코딩

### 3. 시각화 단계
- **탐색적 분석**: matplotlib, seaborn 활용
- **분포 시각화**: 히스토그램, 박스플롯, 바이올린 플롯
- **관계 시각화**: 산점도, 상관관계 히트맵
- **시계열 분석**: 시간에 따른 패턴 분석

### 4. 통계 분석
- **기술통계**: 평균, 분산, 분위수 계산
- **상관분석**: 피어슨, 스피어만 상관계수
- **가설검정**: t-test, chi-square, ANOVA
- **회귀분석**: 선형회귀, 다중회귀 모델링

## 프로젝트 특화 가이드라인

### IQA (Image Quality Assessment) 분석
- **메트릭 평가**: PSNR, SSIM, BRISQUE 등 계산
- **데이터셋 처리**: TID2013, CID2013, LIVE 등 표준 데이터셋
- **점수 분포 분석**: MOS(Mean Opinion Score) 통계 분석
- **모델 성능 평가**: 상관계수 기반 성능 측정

### 크랙 세그멘테이션 분석
- **이미지 전처리**: 크기 조정, 정규화, 데이터 증강
- **마스크 분석**: 크랙 영역 통계, 분포 특성
- **데이터 불균형**: SMOTE, 가중치 조정 기법 적용
- **성능 평가**: IoU, Dice coefficient, confusion matrix

### 통계 분석 패턴
- **연속형 변수**: 히스토그램, 분위수 분석
- **범주형 변수**: 빈도분포표, 카이제곱 검정
- **이변량 분석**: 산점도, 상관관계, 회귀분석
- **다변량 분석**: PCA, 클러스터링, 판별분석

## 출력 형식

### 분석 결과 저장
- **Excel 파일**: 통계 결과, 요약표는 `.xlsx` 형식으로 저장
- **시각화**: PNG 형식으로 고해상도 저장
- **모델**: pickle 형식으로 직렬화 저장
- **보고서**: Jupyter notebook에 마크다운으로 결과 정리

### 파일명 규칙
- 분석 스크립트: `ps{번호}_{프로젝트명}_{분석내용}.py`
- 노트북: `{카테고리}_{분석대상}_{상세설명}.ipynb`
- 결과 파일: `rlt_{분석내용}_{날짜}.xlsx`

## 품질 기준

### 코드 품질
- 함수화: 재사용 가능한 분석 함수 작성
- 문서화: docstring, 주석을 통한 설명 추가
- 오류 처리: try-except를 활용한 예외 처리
- 최적화: 대용량 데이터 처리 시 메모리 효율성 고려

### 분석 품질
- 재현성: 랜덤 시드 고정, 환경 정보 기록
- 검증: 교차 검증, 홀드아웃 세트 활용
- 해석: 통계적 유의성 및 실무적 의미 해석
- 시각화: 명확하고 이해하기 쉬운 그래프 작성
