## 파일명 규칙

```
{섹션번호}_{페이지번호}_{페이지명}.py
```

- **섹션번호**: 사이드바 그룹 정렬 (0, 1, 2, ...)
- **페이지번호**: 섹션 내 순서 (10, 11, 12, ... / 20, 21, ...)
- **`===` 접미사**: 섹션 허브 — 기능 없이 하위 메뉴 안내만 표시
- **`_` 접두사**: `_news_collection/` 등 서브패키지 — 사이드바에 미표시

## 새 페이지 추가 규칙

| # | 작업 | 내용 |
|---|------|------|
| 1 | 파일 생성 | `pages/{섹션}_{번호}_{이름}.py` |
| 2 | 기본 골격 | `check_login_status()` + `display_user_info_header()` |
| 3 | 권한 설정 | 필요 시 `require_admin_permission()` 추가 |
| 4 | 섹션 허브 | 해당 섹션 `===` 파일의 카드 목록에 추가 |

## 섹션 번호 확장 규칙

- 0~4: 핵심 운영 기능
- 5~7: 모니터링/분석
- 8: 개발 문서 (내부용)
- 9: 시스템 설정 (관리자 전용)
- 10+: 필요 시 확장
